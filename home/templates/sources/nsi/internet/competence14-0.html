Qu'est-ce que Bootstrap ?

Bootstrap est un framework open source pour le développement HTML-CSS d'un site. Sa philosophie consiste toujours à séparer le fond (HTML) de la forme (CSS), mais propose des classes toutes faites qu'il suffit d'invoquer dans le code HTML. Cela facilite la compréhension du code HTML et augmente la vitesse de création d'un site, vu que de nombreuses possibilités ont déjà été codées en CSS.

Bootstrap utilise un système de containers dans lesquels il fait évoluer des blocs suivant une structure CSS de type grid ou flex. Par ailleurs, ces blocs se trouvent répartis dans tout l'écran, qui est virtuellement composé de 12 colonnes. Cela facilite la construction responsive du site, qui est devenue indispensable de nos jours.

En résumé, Bootstrap propose une structure de code et des outils afin d'augmenter la productivité du développeur et de rendre le site design responsive dès sa construction.

Boostrap utilise JavaScript, JQuery et des variables Sass pour son fonctionnement mais ne modifie pas le design pattern utilisé pour développer votre site Web.

Comment intégrer Bootstrap à un site

Il suffit de rajouter dans la partie head du site les balises suivantes :

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-T3c6CoIi6uLrA9TneNEoa7RxnatzjcDSCmG1MXxSR1GAsXEV/Dwwykc2MPK8M2HN" crossorigin="anonymous">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-C6RzsynM9kWDrMNeT87bh95OGNyZPhcTNXj1NW7RuBCsyN/o0jlpcV8Qyq46cDfL" crossorigin="anonymous"></script>

Il convient de mettre à jour la dernière version de Bootstrap en remplaçant bootstrap@5.3.2 par le numéro de version le plus récent, qu'on trouve facilement sur le site officiel getbootstrap.com

Les graphes

Un graphe est une structure de données relationnelle, constituée de sommets et de relations. Alors qu'il n'existe qu'un unique sommet dans le cas d'un arbre, il en existe plusieurs dans le cas d'un graphe. Les sommets correspondent donc en quelque sorte aux noeuds des arbres.

La structure d'un graphe diffère de celle d'un arbre en ce sens qu'un arbre part de la racine et se propage des parents vers les enfants, alors qu'un graphe peut évoluer dans tous les sens.

Les graphes sont une forme très utilisée de modélisation d'une situation concrète. On les retrouve par exemple dans les réseaux, la bioinformatique ou les bases de données relationnelles (diagramme UML).

Les graphes non-orientés

Dans un graphe non-orienté les relations sont appelées des arêtes.

A RETENIR ...

Deux arêtes sont adjacentes si elles ont au moins un sommet commun.
Deux sommets sont adjacents si ils sont reliés par une arête.
Le degré d'un sommet est le nombre d'arêtes qui le joignent.
Une chaîne est une suite de sommets adjacents.
Une chaîne est élémentaire si elle ne comporte qu'une seule fois un même sommet.
Un cycle est une chaîne dont le sommet de départ est le même que le sommet d'arrivée.
Un graphe est connexe, si pour tout couple de sommets on peut trouver une chaîne les reliant. Les graphes connexes sont donc construits d'un seul morceau. Les graphes non connexes sont construits en composantes connexes, c'est à dire en sous-graphes connexes. Un graphe est complet si chacun de ses sommets est directement reliés à tous les autres sommets.

Les graphes orientés

Dans un graphe orienté les relations sont appelées des arcs.

A RETENIR ...

Deux arcs sont adjacents si ils ont au moins un sommet commun.
Deux sommets sont adjacents si ils sont reliés par un arc.
Le degré d'un sommet est le nombre d'arcs dont il est l'extrêmité.
Un chemin est une suite de sommets adjacents.
Un chemin est élémentaire si il ne comporte qu'une seule fois un même sommet.
Un circuit est un chemin dont le sommet de départ est le même que le sommet d'arrivée.
Un graphe est fortement connexe, si pour tout couple de sommets on peut trouver un chemin allant de l'un vers l'autre et un chemin inverse. Les graphes non fortement connexes sont construits en composantes fortement connexes, c'est à dire en sous-graphes fortement connexes. Un graphe est complet si chacun de ses sommets est directement reliés à tous les autres sommets.

Graphes pondérés

Un graphe étiqueté est tel que chaque relation est affectée d'un symbole. Un graphe pondéré est un cas particulier de graphe étiqueté dans lequel les symboles sont des nombres positifs appelés poids.

Les graphes pondérés peuvent être utilisés pour les calculs de trajets entre des villes. Dans ce cas, les différents poids peuvent représenter les distances en km ou les temps de parcours en min.

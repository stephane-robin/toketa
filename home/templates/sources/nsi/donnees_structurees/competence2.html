Qu'est-ce qu'une base de données ?

La vidéo suivante publiée par Fun Mooc sur YouTube détaille l'utilisation des données structurées :
🎧 vidéo | Données comment les manipuler - Fun Mooc YouTube

L'informatique a pour vocation le traitement, le stockage et l'affichage de l'information. Celle-ci peut être regroupée sous la forme de tables connectées entre elles par des relations. Chaque table contient des données qui occupent un espace mémoire dont le système de gestion connaît l'adresse. L'ensemble de ces tables permet de décrire une situation ou un objet et constitue une base de données.

VOCABULAIRE ...

Une donnée est une valeur associée à un objet, un individu, un événement permettant de le décrire.

Une base de données est un ensemble de données reliées entre elles et structurées avec des tables.

Le Big Data correspond à une quantité importante de données générées par l'utilisation de technologies numériques.

Il existe 3 types de données :

  • les données quantitatives, qui s'expriment sous forme de nombres,
  • les données qualitatives, qui ne sont pas quantifiables et s'expriment sous forme de texte,
  • les données catégorielles, qui peuvent être rangées dans des catégories définies à l'avance. Leur valeur peut être numérique ou textuelle.

Utilité des bases de données

Tant qu'un programme est en cours d'exécution, il conserve en mémoire les valeurs des variables utilisées. En revanche, une fois qu'on stoppe ce programme ou qu'on se déconnecte d'un site Internet, les valeurs des variables sont perduées. Certaines ont pour vocation d'être temporaires, mais d'autres comme le montant du compte en banque, le niveau de jeu atteint, le mot de passe de connexion, ... doivent être conservées dans un serveur distant dont le rôle unique est de stocker des bases de données.

On vient de voir que les bases de données servent de support à vos programmes. Mais l'inverse se produit également. On peut construire des programmes capables d'optimiser la recherche d'information dans une base de données, ou encore qui fournissent un modèle d'intelligence artificielle pour la situation décrite par la base de données.

Elements caractéristiques des bases de données

Les bases de données sont constituées de tables, elles-mêmes composées de descripteurs (les colonnes) et d'enregistrements, ou entrées (les lignes). Chaque élément d'une table est appelé un champ et dans le cas de l'utilisation d'un tableur Excel une cellule. Chaque table contient une clé primaire permettant d'identifier de manière unique chaque enregistrement. Par ailleurs, les différentes tables d'une base de données peuvent être reliées entre elles à l'aide de clés étrangères.

Attention, un tableur Excel n'est pas une base de données même s'il se présente sous forme de tableau. Il s'agit en fait d'une application qui facilite le traitement et l'affichage de données organisées en tables.

Voici un exemple de table d'une base de données, qui ressemble assez à un tableur Excel :

IDnomhabitants

4153Geneve198 979
312465Bern133 115
213Luzern81 592

Ces données sont accessibles pour consultation et modification à l'aide d'un logiciel appelé SGDB Système de Gestion de Bases de Données.

On retrouve une clé primaire (ID), des descripteurs (ID, nom, habitants), 4 entrées car 4 lignes représentant des données, et par exemple un champ "Geneve".

Les métadonnées d'une image sont facilement accessibles, en sélectionnant l'image puis en cherchant ses Propriétés (souvent avec un clic droit de souris).

Les métadonnées d'une page Web, quant à elles, sont contenues dans la balise <head> :

<head>
  <title> Toketa-Classmate </title>
  <link rel='stylesheet' href='style_classroom1.css"; ?>'>
  <link rel='icon'
        type='image/png'
        href='images/general/symbole.png"; ?>'>
  <meta charset='utf-8'>
</head>

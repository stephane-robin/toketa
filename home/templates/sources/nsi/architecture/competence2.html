Nous allons explorer le Terminal et apprendre à l'utiliser en suivant des directives étape par étape.

Toutes les instructions nécessaires à ce travail pratique se trouve dans le chapitre Le système Linux.

  • Ouvrir le Terminal puis se rendre dans le dossier Documents,
  • Afficher tous les fichiers et sous-dossiers présents dans Documents, avec les détails et les fichiers cachés.
  • Creer un sous dossier appelé Dossier_term
  • Se rendre dans Dossier_term, puis y créer un fichier vide appelé fichier1.txt
  • Ouvrir fichier1.txt avec nano, puis écrire dans ce fichier le texte "Les sanglots longs de l'automne blessent mon coeur d'une langueur monotomne."
  • Quitter nano, puis lire le contenu de fichier1.txt dans le Terminal
  • Copier fichier1.txt puis le coller dans Documents sous le nom fichier2.txt
  • Ouvrir fichier2.txt avec nano, puis en modifier le contenu pour que soit écrit print("Les sanglots longs de l'automne ...")
  • Renommer fichier2.txt en fichier3.py en respectant bien le changement d'extension.
  • Exécuter en Python fichier3.px

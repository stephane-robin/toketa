Programmation Python

Bien débuter en Python

Python est un langage de programmation interprété dont la philosophie repose sur la simplicité du code, la liberté du programmeur et l'implémentation de bonnes pratiques. Rien n'est fait pour empêcher le programmeur de mener une action, même si celle-ci semble inappropriée.

Nous découvrirons les bases de Python à l'aide d'exemples progressifs qui vous apprendront à manipuler les chaînes de caractères, les conditions, les boucles, les fonctions, ...

Progresser en Python

Progresser en programmation nécessite de bien comprendre les bases de la structure de code mais également comment manipuler les données générées par ce code. Le meilleur moyen d'y parvenir reste encore de s'entraîner en codant toujours plus.

Nous apprendrons à manipuler des listes, des dictionnaires et à construire des codes plus complexes. Nous découvrirons comment tester nos codes et gérer nos erreurs. Nous apprendrons à utiliser nos premières bases de données.

Python avancé

Il est important de choisir le paradigme de programmation le plus adapté au travail que nous voulons fournir. La programmation objet est une manière de programmer qui utilise un modèle proche de la réalité où on représente chaque élément en tant qu'objet au sein d'une classe. Chaque objet a des caractéristiques propres qu'on peut modifier.

Nous apprendrons à programmer en orienté objet, nous utiliserons également la récursivité pour effectuer des calculs complexes, nous découvrirons les bases de la cryptographie.

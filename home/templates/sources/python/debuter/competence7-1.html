La bibliothèque time

Pour connaître le temps écoulé en secondes depuis la création de l'informatique (1er janvier 1970) :

import time
temps = time.time()

Cela permet notamment de connaître le temps d'exécution d'un bloc d'instructions en mesurant le temps avant l'exécution puis le temps après l'exécution :

import time
debut = time.time()
i = 0
while i < 100000:
    i += 1
print(i)
fin = time.time()

duree = fin - debut

Pour mettre en pause l'exécution d'un programme pendant 10 secondes :

import time
print("Première phrase")
time.sleep(10)
print("Deuxième phrase après la pause")

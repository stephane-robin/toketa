La boucle while permet de répéter un bloc d'instructions tant qu'une condition est remplie. On utilise donc la boucle while lorsqu'on ne connaît pas le nombre de répétitions à effectuer et que celui-ci dépend de l'évolution de la situation (par exemple on continue de jouer tant que le nombre de vies d'un joueur n'est pas nul). Dans le cas où l'on connaît à l'avance le nombre de répétitions à effectuer, on utilise préférentiellement une boucle for, qui sera étudiée plus tard.

On peut répéter Continuez de jouer tant que la variable nbreVies est positive

nbreVies = 3
while nbreVies > 0:
    print("Continuez de jouer")
    nbreVies = nbreVies - 1
print("Fin du jeu")

Si on détaille ce que fait le programme, on obtient ceci :

nbreVies prend la valeur 3
3 est strictement positif, donc on exécute ce qui se trouve dans la boucle while {
    on affiche "Continuez de jouer"
    nbreVies prend la valeur 2
    }
2 est strictement positif, donc on exécute ce qui se trouve dans la boucle while {
    on affiche "Continuez de jouer"
    nbreVies prend la valeur 1
    }
1 est strictement positif, donc on exécute ce qui se trouve dans la boucle while {
    on affiche "Continuez de jouer"
    nbreVies prend la valeur 0
    }
0 n'est pas strictement positif, donc on n'exécute plus ce qui se trouve dans la boucle while 
on continue l'exécution du reste du programme, donc on affiche "Fin du jeu"

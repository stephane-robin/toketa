Il est souvent nécessaire de créer des blocs de codes qu'on appelle à chaque fois qu'on en a besoin. Cela évite les répétitions inutiles de code. Ces blocs réutilisables sont des fonctions, qui effectuent un travail à partir de données connues au départ (les paramètres) pour modifier ces données ou fournir un résultat à l'arrivée. Ainsi, les programmes sont souvent constitués de fonctions qui s'appellent les unes les autres pour accomplir une tâche.

Il existe une réelle analogie avec les fonctions qu'on étudie en mathématiques, qui prenne x comme paramètre, lui font subir des calculs pour renvoyer une valeur notée f(x).

Pour utiliser une fonction dans un programme, il faut tout d'abord la définir (pour que le programme la connaisse), puis il faut l'appeler (pour que le programme l'exécute dans un cas particulier).

Pour reprendre notre analogie avec les mathématiques, on définit d'abord une fonction par f(x) = 2 x + 1, puis on l'appelle en calculant par exemple la valeur de f(3).

Pour définir une fonction en Python, rien de plus simple, il faut :

  • utiliser le mot-clé def

  • choisir les paramètres de la fonction, qui vont faire varier le résultat, et les placer entre parenthèses

  • penser à indenter les instructions afin de créer un bloc, qui définit pour Python le début et la fin de la fonction

  • si une valeur est renvoyée à la fin, ne pas oublier le mot clé return

Par exemple la fonction saluer() qui dit bonjour à la personne dont le nom est entré en paramètre se définit de la façon suivante en Python :

def saluer(nom):
    return "bonjour " + nom

Voici un autre exemple, une fonction qui retire un point au joueur et qu'on appelera uniquement lorsque le joueur perd :

def perdreUnPoint(nbrePoints):
    nbrePoints = nbrePoints - 1
    print("Vous venez de perdre un point")
    return nbrePoints

On peut bien évidemment utiliser une fonction Python pour définir une fonction mathématique, comme dans l'exemple ci-dessous (il faut alors importer la bibliothèque math qui met à disposition certaines fonctions mathématiques) :

from math import *

def f(x):  # on definit la fonction racine carre de x + 1
    return sqrt(x + 1)

Notons qu'il est tout à fait possible de définir une fonction sans argument, ou une fonction qui ne renvoie pas de valeur mais se contente d'exécuter une tâche. Dans ce cas, il faut quand même utiliser les () pour que Python ne confonde pas le nom de la fonction avec le nom d'une variable. On parle alors de procédure. En voici un exemple 

def annoncer():
    print("Cet été la météo sera clémente")
    print("Pensez à acheter de la crème solaire")

A vous de jouer : écrire une fonction mathématique f définie par f(x) = 5 x - 2. Calculer ensuite puis afficher la valeur de f(3)

Solution : 

def f(x):
    return 5 * x - 2

print(f(3))

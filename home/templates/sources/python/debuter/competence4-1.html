Dans le cas d'une chaîne de caractères, l'opérateur + ne sert pas à additionner, mais à coller un texte à la suite d'un autre. On parle de concaténation pour cette opération. Par exemple 

print("Les oranges sont " + "des fruits")

affichera Les oranges sont des fruits sans procéder à aucun calcul.

A toi de jouer :

Pour bien comprendre la différence entre l'addition et la concaténation, exécute le code suivant :

a = 12
b = 15
print(a + b) # addition de nombres, affiche 27

a = "12"
b = "15"
print(a + b) # concaténation de caractères, affiche "1215"

La duplication

Dans le cas d'une chaîne de caractères, l'opérateur * permet de répéter un texte, comme dans l'exemple ci-dessous :

print("Les oranges sont des fruits" * 3)

Les f-string

Il s'agit d'un format particulier de string qui tient compte de la valeur des variables inclues dans la chaîne de caractères, comme dans l'exemple ci-dessous :

resultat = f"Je m'appelle {nom} et j'ai {age} ans"

Les f-string sont particulièrement pratiques car il n'est pas nécessaire de modifier le type des variables inclues dans la chaîne de caractères pour assurer leur affichage. Dans l'exemple ci-dessus il n'est pas nécessaire de transformer age en str en utilisant l'instruction str(age).

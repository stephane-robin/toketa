La boucle while permet de répéter un bloc d'instructions tant qu'une condition est remplie. On utilise donc la boucle while lorsqu'on ne connaît pas le nombre de répétitions à effectuer et que celui-ci dépend de l'évolution de la situation (par exemple on continue de jouer tant que le nombre de vies d'un joueur n'est pas nul). Dans le cas où l'on connaît à l'avance le nombre de répétitions à effectuer, on utilise préférentiellement une boucle for, qui sera étudiée plus tard.

Structure d'une boucle while

Une boucle while se construit avec :

  • une condition qui renvoie True ou False,
  • une indentation pour indiquer le bloc de code à répéter,
  • une variable dont la valeur change pour modifier la condition au tour suivant.

Dans l'exemple ci-dessus, la condition nbreVies > 0 renvoie d'abord True puisque nbreVies vaut 3, puis cette condition est testée à chaque tour jusqu'à ce que nbreVies soit égal à 0. Dans ce cas la condition nbreVies > 0 renvoie False, le programme n'effectue pas de tour supplémentaire et exécute le code placé après la boucle while (après l'indentation).

On renarque que c'est la variable nbreVies qui varie pour modifier la condition à tester.

Répéter une action avec une boucle while

Il est possible de répéter une action avec une boucle while, même si on préfère en général utiliser une boucle for dans ce cas, ce que nous étudierons plus tard. Pour cela, il faut créer un compteur qu'on fait varier, et il faut comparer ce compteur à une certaine valeur pour savoir quand s'arrêter.

A vous de jouer : testez le code suivant qui permet de compter jusqu'à 10 :

compteur = 1

while compteur < 11:
    print(compteur)
    compteur = compteur + 1

On sait maintenant compter jusqu'à 10 en utilisant une boucle while. Mais comment faire pour sauter un chiffre, par exemple pour ne pas afficher le chiffre 6 ?

A vous de jouer : essayez maintenant d'écrire un code utilisant un boucle while, qui compte jusqu'à 10 en affichant ces nombres sur une même ligne, et sans afficher 6.

compteur = 1

        print(compteur)

Notons qu'on peut également utiliser l'instruction continue, qui permet de sauter un tour de boucle.

compteur = 1

while compteur < 11:

    if compteur == 6:
        compteur = compteur + 1
        continue

    print(compteur, end=" ")
    compteur = compteur + 1

Le risque avec une boucle while est que la condition d'arrêt ne soit jamais remplie, ce qui ne permet pas à la boucle de se terminer. On crée ainsi une boucle infinie qui bloque le programme. Par exemple 

compteur = 10
while compteur >= 0:
    print("le compteur est positif")

N'exécutez pas le code ci-dessus car la variable compteur sera toujours supérieure à 0 (elle vaut 10) et le programme ne se terminera jamais. Certains langages comme PHP ne permettent pas de dépasser un certain nombre de tours, mais ce n'est pas le cas de Python qui continue à travailler tant qu'on ne l'arrête pas. Il faut dans ce cas appuyer sur la touche Stop de l'environnement de travail ou Ctrl+C au terminal pour arrêter manuellement l'exécution du programme.

    compteur = compteur + 1

Dans le code ci-dessus on a compté le nombre de caractères de maChaine pour obtenir 7. Le programme parcourt donc la chaine de caractères en partant de l'indice 0, jusqu'à l'indice 6 inclus, qui correspond à l'indice du dernier caractère. En fait, il faut laisser faire ce travail à l'ordinateur en utilisant l'instruction len(maChaine) qui renvoie la longueur de cette chaîne. Par ailleurs, on a tendance à appeler indice le compteur des lettres un indice et de ce fait utiliser i pour le compteur. Ainsi le code précédent peut être améliorer de la façon suivante :

mot = "bonjour"
i = 0

while i < len(mot):
    print(mot[i])
    i = i + 1

On sait maintenant récupérer tous les caractères d'une chaîne pour les afficher.

Travail pratique avec la méthode replace()

Essayons autre chose de plus compliqué. Par exemple imaginons qu'on veuille changer les "o" en "a" dans un texte. On ne va pas le faire manuellement car cela pourrait être très long et fastidieux avec un texte de 1000 lignes 😓 Il faut donc créer un programme qui soit capable de le faire pour nous.

Nous savons déjà que pour changer une lettre en une autre lettre à l'intérieur d'un mot, il existe la méthode mot.replace("o", "a")

A vous de jouer : exécutez le code ci-dessus et affichez la valeur de nouveau_mot, puis comparez le résultat obtenu en exécutant le code suivant :

mot = "bonjour"
nouveau_mot = mot.replace("o", "a")
print(mot)
print(nouveau_mot)

Notez que la méthode replace() ne modifie pas la variable initiale mot, mais elle crée, à partir de cette variable mot, une valeur qui sera stockée dans une autre variable nouveau_mot.

Commençons notre travail petit à petit avec un seul mot plutôt qu'un long texte. Pour cela, définissons la variable mot = "bonjour". Les chaînes de caractères n'acceptent pas le changement facilement. Il faut soit changer tout le texte en une seule fois, soit reconstruire le nouveau mot étape par étape, c'est à dire caractère par caractère. On pourrait écrire mot = "banjaur", mais dans ce cas, c'est nous qui ferions tout le travail et non pas le programme.

Puisque dans le mot "bonjour", les "o" ont pour indice 1 et 4 (car l'indice du 1er caractère vaut 0), on sait que mot[1] et mot[4] valent "o". En théorie, il suffirait donc d'écrire mot[1] = "a" et mot[4] = "a" pour obtenir satisfaction. Et naturellement on a envie d'écrire le code suivant :

mot = "bonjour"
mot[1] = "a" # erreur
mot[4] = "a" # erreur
print(mot)

Malheureusement, ce n'est pas possible car mot[1] et mot[4] ne sont pas des variables autonomes. Le code ci-dessus déclenche donc une erreur.

Il faut en fait reconstruire tout le mot en rajoutant chaque caractère un à un. On utilise ce qu'on appelle la concaténation pour placer un "a" lorsque le caractère testé est un "o" et ne rien changer pour les autres lettres.

mot = "bonjour"
nouveau_mot = ""
i = 0

while i < len(mot):
    if mot[i] == "o":
        nouveau_mot = nouveau_mot + "a"
    else:
        nouveau_mot = nouveau_mot + mot[i]
    i = i + 1
    
print(nouveau_mot)

Que fait notre code ci-dessus ?

On crée tout d'abord une chaîne de caractères vide qui s'appelle nouveau_mot, et qui va contenir notre mot modifié. Retenons que pour créer une chaîne de caractères vide, il faut utiliser l'instruction nouveau_mot = "".

On parcourt ensuite notre mot caractère par caractère avec une boucle while, puis à l'aide d'une condition if, on rajoute "a" lorsque le caractère lu est "o", on rajoute la lettre normalement prévue sinon.

Retenez donc cette stratégie, qui est utile chaque fois qu'on cherche à modifier la valeur d'un mot.

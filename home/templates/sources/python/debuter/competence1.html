Découvrir les variables par la pratique

Ecoute la vidéo ci-dessous puis télécharge et complète le Jupyter Notebook suivant pour découvrir les bases des variables en Python. Rappel, il est possible d'ouvrir un Jupyter Notebook sur le site basthon.fr.

L'essentiel à connaître

Qu'est-ce qu'une variable ?

Une variable est un objet informatique composé d'un nom, d'une valeur, d'une adresse et d'un type.

  • le nom doit être explicite, ne peut pas être composé d'une instruction Python, ne peut pas commencer par un chiffre
  • la valeur d'une variable est amenée à changer au cours de l'exécution d'un programme
  • l'adresse de la variable permet de la retrouver en mémoire
  • le type correspond à la famille de cette variable et permet de lui fournir une structure qui répond à ses besoins

Comment définir une variable ?

On définit la valeur d'une variable avec un signe = comme dans l'exemple ci-dessous :

age = 16

On pourra par la suite lui donner de nouvelles valeurs autant de fois que nécessaire.

Comment utiliser une variable ?

On utilise une variable en l'appelant par son nom et le programme remplacera ce nom par sa valeur, comme dans l'exemple ci-dessous où identite prend la valeur Antoine - Robert :

prenom = "Antoine"
nom = "Robert"
identite = nom + " - " + prenom

Les principales opérations

  • + pour l'addition
  • - pour la soustraction
  • * pour la multiplication
  • ** pour la puissance
  • / pour la division
  • // pour le quotient de division
  • % pour le reste de division

Comment afficher une variable ?

On utilise l'instruction print() et le nom de la variable. Par exemple, le code suivant affiche Hervé a 17 ans :

prenom = "Hervé"
age = "17"
print(prenom + " a " + age + " ans")

Comment saisir la valeur d'une variable au clavier

On utilise l'instruction input() et on stocke la valeur saisie dans une variable, comme dans l'exemple ci-dessous qui affiche Quel est ton nom ? et attend une réponse avant de la stocker dans la variable nom :

nom = input("Quel est ton nom ? ")

Comment définir une variable

Une variable est un peu comme un carton de rangement. On a besoin d'un nom pour savoir ce qu'il contient, et bien sûr on doit pouvoir accéder à son contenu quand on en a besoin. On doit même pouvoir retirer une affaire du carton, rajouter une affaire ou en modifier totalement le contenu. Afin d'utiliser la taille de carton la plus appropriée à son contenu, on a besoin de connaître le type de carton nécessaire. Il est également important de connaître son emplacement dans la maison lorsqu'on a besoin d'y accéder. Une variable est construite suivant le même principe qu'un carton, et dispose des mêmes caractéristiques : un nom, une valeur, un type et une adresse mémoire.

Une variable permet de stocker une donnée du code. Par exemple dans un jeu, on a besoin de garder en mémoire le score du joueur, sur un site internet on a besoin de garder en mémoire le pseudo de l'utilisateur lorsqu'il est identifié. On comprend bien que les variables sont les briques les plus essentielles au fonctionnement de nos programmes.

En Python, on ne s'occupera que du nom de la variable, de sa valeur et de son type. La gestion de la mémoire n'est pas du ressort du programmeur, contrairement au langage C. Elle est automatisée en Python par un garbage collector. Cela facilite le travail du développeur mais n'optimise pas l'utilisation de la mémoire et ralentit l'exécution du code.

Voici un premier exemple où le nom de la variable est age :

age = 18

On utilise le signe "=" pour donner la valeur 18 à la variable appelée age. On peut maintenant changer la valeur de age autant de fois que l'on souhaite. Celle-ci est stockée en mémoire sous le nom age. On parle de variable justement parce qu'on peut en changer la valeur autant de fois que nécessaire.

⚡ Attention, en mathématiques le signe "=" est réversible, c'est à dire qu'on peut l'utiliser dans les deux sens et on peut également résoudre une équation pour obtenir la valeur d'une inconnue x par exemple. En informatique, c'est beaucoup plus simple, le signe "=" signifie uniquement qu'on donne une valeur à une variable. Il n'y a pas de notion d'égalité. Pour mieux comprendre, analysons le code ci-dessous :

x = 12
y = x
x = 44

Un programme se lit et s'exécute ligne par ligne. Dans ce cas,

  • x prend la valeur 12
  • y prend la valeur de x c'est à dire 12
  • x prend la valeur 44 et y conserve sa valeur 12. Bien qu'à la ligne précédent il était écrit y = x, les deux variables n'ont pas la même valeur à la fin du code. IL NE S'AGIT PAS D'UNE EGALITE MAIS DE L'AFFECTATION DE VALEURS A DES VARIABLES.

A RETENIR ...

Le signe = ne symbolise pas une égalité en Python, mais l'affectation d'une valeur à une variable.

Comment nommer une variable

Le nom d'une variable ne peut pas commencer par un chiffre, ne peut pas contenir d'espace, ne peut pas contenir de caractère spécial (#!.;()][/\<>) et ne peut pas être un mot-clé du langage Python comme par exemple print, if, else, while, def, int, str, float, bool, for, range, and, or, elif, input, match, not, chr, ord, list, ....

Comment utiliser une variable

On utilise une variable en se servant de son nom à l'intérieur d'instructions Python.

Il est par exemple possible de faire des calculs en utilisant des opérations qui servent à modifier la valeur des variables. Les virgules servent de séparateurs en Python et il faut utiliser un point pour séparer la valeur entière de la valeur décimale dans un nombre réel, exactement comme dans la langue anglaise. Par exemple, pour affecter la valeur 17,2 à la variable x, on écrit

x = 17.2

Pour afficher la valeur de x, il suffit maintenant d'utiliser l'instruction :

print(x)

qui affichera 17.2.

Pour les nombres entiers (int) et les nombres réels (float), on peut utiliser les opérations suivantes :

+ pour l'addition

L'instruction ci-dessous donne la valeur 8 à la variable x.

x = 5 + 3

- pour la soustraction

L'instruction ci-dessous donne la valeur 3 à la variable x.

x = 5 - 2

* pour la multiplication

L'instruction ci-dessous donne la valeur 35 à la variable x.

x = 5 * 7

/ pour la division

L'instruction ci-dessous donne la valeur 1.4 à la variable x.

x = 7 / 5

** pour l'exposant

L'instruction ci-dessous donne la valeur 49 à la variable x.

x = 7 ** 2

// pour le quotient d'une division

Il est également possible d'effectuer une division euclidienne. Par exemple, on sait que \(7 = 1 \times 5 + 2\), donc le quotient de cette opération vaut 1 et le reste vaut 2.

L'instruction ci-dessous donne la valeur 1 à la variable x.

x = 7 // 5

Attention la division décimale de 7 par 5 donne 1.4, mais la division euclidienne de 7 par 5 donne 1 fois 5 plus 2. Ainsi le quotient vaut 1 et le reste de division vaut 2, ce qui explique le résultat obtenu.

% pour le reste de division

L'instruction ci-dessous donne la valeur 2 à la variable x. On utilise encore une fois la division euclidienne. Celle-ci permet de mettre en valeur l'opérateur modulo % pour savoir si un nombre est multiple d'un autre, ou si une séquence de caractères peut être regroupée en blocs de 8 éléments.

x = 7 % 5

Les différents types de variables

Chaque variable appartient à une famille particulière relative à la valeur de cette variable, appelée type. Les types primaires en Python sont les suivants:

  • les nombres entiers sont stockés dans des variables de type int pour integer
  • les nombres réels sont stockés dans des variables de type float pour nombres à virgules flottantes
  • les textes (chaînes de caractères) sont stockés dans des variables de type str pour string
  • les valeurs True et False sont stockées dans des variables de type bool pour booléens.

La taille de l'espace occupé par une variable en mémoire dépend fortement du type de cette variable. C'est la raison pour laquelle il est parfois préférable de travailler avec un type de variable plutôt qu'un autre. Par exemple les entiers occupent moins de place que les nombres réels qui nécessitent un espace mémoire avant la virgule et un espace mémoire après la virgule.

On peut afficher le type d'une variable avec l'instruction type() et le nom de la variable. Par exemple : 

print(type(age))

affichera int.

Comment manipuler les variables

Inverser deux variables

En Python, on peut définir plusieurs variables sur une même ligne, en les séparant par une virgule. Par exemple :

nom, prenom = "Gulliver", "Cedric"

On veut maintenant inverser les valeurs de deux variables. En fait, ce n'est pas si simple. Pour s'en rendre compte, il suffit d'imaginer deux fichiers Word appelés monFichier1.docx et monFichier2.docx. Si on veut changer le nom de monFichier1.docx en monFichier2.docx, le système d'exploitation de la machine refuse cette modification car on ne peut pas avoir deux fichiers du même nom dans un même dossier.

Si on bouge monFichier2.docx dans un autre dossier, qu'on renomme monFichier1.docx, cela va marcher mais on ne pourra plus faire revenir monFichier2.docx dans son dossier initial puisqu'on aurait dans ce cas deux fichiers du même nom.

Il faut en fait utiliser un nom de fichier temporaire pour monFichier1.docx, par exemple l'appeler temp.docx. On peut ensuite renommer monFichier2.docx en monFichier1.docx et temp.docx en monFichier2.docx.

Effectuer des calculs avec des variables

On peut calculer la valeur d'une variable en utilisant celle d'autres variables. Par exemple si on veut trouver la hauteur d'un muret composé de pots surmontés d'arbustes, on peut effectuer le calcul suivant

hauteur_pots = 0.6
hauteur_arbustes = 1.2
hauteur_muret = hauteur_pots + hauteur_arbustes

L'avantage d'utiliser des variables plutôt que des valeurs directement est le même que celui du calcul littéral en mathématiques. Cela rend le calcul plus facile lorsqu'on doit changer les valeurs initiales.

A toi de jouer : écris un script qui calcule la vitesse moyenne d'un cycliste, connaissant la distance de montée de 16 km, la distance de descente de 51 km, le temps de montée de 1,2 heures et le temps de descente de 1,5 heures.
Indication : la vitesse moyenne se calcule en faisant la moyenne de la vitesse de montée et de la vitesse de descente. Une vitesse est le quotient d'une distance par un temps, d'où l'unité km/h.

On peut tout à fait utiliser Python comme une simple calculatrice. Dans ce cas, il suffit d'écrire les lignes suivantes pour notre programme :

vitesse_moyenne = (16 / 1.2 + 51 / 1.5) / 2
print(vitesse_moyenne)

Toutefois, il est souvent astucieux de créer plusieurs variables et de détailler nos calculs afin de pouvoir réutiliser des portions de code ou les variables dans d'autres parties du programme. On pourrait donc écrire :

distance_montee = 16 # en km
temps_montee = 1.2 # en heures
distance_descente = 51
temps_descente = 1.5

vitesse_montee = distance_montee / temps_montee
vitesse_descente = distance_descente / temps_descente

vitesse_moyenne = (vitesse_montee + vitesse_descente) / 2
print(vitesse_moyenne)

Affichage de variables dans un texte

Nous allons voir comment afficher les valeurs de plusieurs variables, ou la valeur d'une variable au sein d'un texte.

nom = "Parker"
prenom = "Peter"
print(prenom + nom) # affiche PeterParker
print(prenom + " - " + nom) # affiche Peter - Parker
print(prenom + "\t" + nom) # affiche Peter   Parker séparés par une tabulation
print(prenom + "\n" + nom) # affiche Peter Parker l'un en dessous de l'autre

On remarque donc que l'instruction print() saute automatiquement la ligne, une fois le travail effectué. Elle correspond en fait au println() qu'on retrouve dans d'autres langages.

Affichage respectant un format

On a vu ensemble qu'on pouvait afficher la valeur d'une variable au milieu d'un texte, en construisant un texte qui contient cette variable. Par exemple le code suivant :

prenom = "Antoine"
date_naissance = "17-02-2008
print(prenom + " est né le " + date_naissance)

affichera le résultat suivant :

Antoine est né le 17-02-2008

Il existe une façon plus efficace d'afficher le même résultat en utilisant l'instruction print(f) :

prenom = "Antoine"
date_naissance = "17-02-2008
print(f"{prenom} est né le {date_naissance}")

Affichage d'une variable de type différent de str

Imaginons qu'on veuille afficher le score d'un joueur à la fin d'une partie. En imaginant qu'il ait obtenu le score de 147 et qu'il s'appelle "Bastien", on serait tempté d'écrire :

prenom = "Bastien"
score = 147

print("Le joueur " + prenom + " a obtenu le score de " + score)

Si tu exécutes ce code, tu te rendras vite compte qu'il déclenche une erreur, car prenom est une chaine de caractères et s'intègre donc parfaitement au reste du texte, mais score est un entier et ne sait pas comment faire pour compléter un texte. Il faut donc le transformer en chaîne de caractères avec l'instruction str et modifier notre code comme ceci :

prenom = "Bastien"
score = 147

print("Le joeur " + prenom + " a obtenu le score de " + str(score))

Il est également possible d'utiliser l'affichage formaté qui est tout à fait adapté à l'affichage de nombres :

prenom = "Bastien"
score = 147

print(f"Le joueur {prenom} a obtenu le score de {score})

Comment saisir une variable au clavier

Il est important de pouvoir communiquer avec l'ordinateur, pour lire des résultats ou donner des informations au programme. Par exemple lorsque vous jouez à votre jeu préféré sur votre smartphone, vous effectuez cliquez sur votre écran pour indiquer au programme vos choix, vous saisissez également votre identifiant au clavier, puis vous observez à l'écran les progrès de votre personnage.

Afin que le programme puisse tenir compte des données de l'utilisateur, il est nécessaire de pouvoir récupérer ce que celui-ci tape au clavier. Une question posée à l'utilisateur et qui appelle une réponse de sa part, doit être insérée dans une instruction input. Il est inutile dans ce cas d'utiliser l'affichage de la question avec print. Ainsi, le code suivant affiche la question Comment vous appelez-vous ? et attend une réponse : 

input("Comment vous appelez-vous ?")

L'interpréteur Python est maintenant en pause jusqu'à ce que l'utilisateur saisisse au clavier sa réponse puis valide avec la touche Entrée.

Attention, cette réponse est perdue si elle n'est pas affectée à une variable. Ainsi, le code ci-dessus doit être modifié de la façon suivante pour être utile :

nom = input("Comment vous appelez-vous ?")

Maintenant seulement, la valeur saisie au clavier par l'utilisateur sera réutilisable lorsqu'on appelle la variable nom, commme par exemple avec le code ci-dessous :

print(nom)

A toi de jouer : demande à l'utilisateur son âge puis affichez la valeur saisie en utilisant une variable age.

age = input("quel est votre âge ?")
print(age)

Quel est le type par défaut saisi au clavier ?

Tout ce que saisit l'utilisateur au clavier est enregistré à l'aide de l'instruction input en tant que chaîne de caractères (texte), même un nombre. Essayez pour vous en convaincre le script suivant :

age = input("quel est ton âge ? ")
print(type(age))

Dans le script ci-dessus la variable age est de type class 'str', c'est à dire chaîne de caractères. L'ordinateur considère que le nombre saisi est en fait du texte.

Mais alors comment faire des calculs avec du texte ? 😓

Afin de pouvoir faire subir des opérations à une variable, il faut que celle-ci soit un nombre, donc principalement de type int (entier) ou de type float (réel). Or, lorsqu'un utilisateur saisit un nombre au clavier, celui-ci est conservé en tant que str (chaîne de caractères). Il faut donc auparavant le transformer en nombre pour pouvoir l'utiliser. On parle de caster une variable pour dire qu'on lui change de type. Pour cela, il suffit d'utiliser devant l'instruction input, l'instruction int() ou float(), comme dans le code suivant :

age = int(input("quel est ton âge ? "))
print(type(age))

Attention toutefois, on ne peut transformer une chaîne de caractères en nombres que si celle-ci est composée exclusivement de nombres. Ainsi, l'instruction int("bonjour") entraînera une erreur et ne sera pas transformé en entier. De même, l'instruction int("12Hervé") entraînera une erreur et ne sera pas transformé en entier.

A RETENIR ...

"2" est une chaîne de caractères composée du symbole deux qu'on ne peut pas soustraire, ou additionner, ... En revanche 2 est un nombre auquel on peut faire subir des opérations. Pour s'en convaincre, essayez l'instruction suivante.

print("2"+"1") # affiche 21
print(2+1) # affiche 3

A toi de jouer : imaginons que le programme souhaite connaître l'âge de l'utilisateur puis que plus tard, il souhaite lui rajouter un an (lors de son anniversaire). Ecrire le programme qui effectue ces instructions.

L'instruction suivante ne renverrai aucune erreur mais conduirait à un résultat absurde :

age = input("quel est votre âge ?")
age = age + 1
print(age)

qui affiche si l'utilisateur saisit la valeur 18 en réponse

>>> 181

Dans ce cas, la valeur 18 saisie possède par défaut le type str et lorsqu'on lui rajoute 1, ce dernier se place à la suite de 18 et n'effectue aucun calcul. On parle de concaténation.

Pour corriger ce problème et effectuer une addition, il faut utiliser le code suivant :

age = int(input("quel est votre âge ?"))
age = age + 1
print(age)

qui fournit le résultat escompté, à savoir :

>>> 19

Il existe en Python un moyen de vérifier qu'une valeur saisie peut être transformée en entier, à l'aide de l'instruction isnumeric().

A toi de jouer : essaie le code ci-dessous pour bien comprendre ce que teste l'instruction isnumeric().

print("22".isnumeric()) # affiche True
print("bonjour".isnumeric()) # affiche False, bonjour est une chaîne de caractères
print("2.45".isnumeric()) # affiche False, 2.45 est un nombre réel
print("-3".isnumeric()) # affiche False, -3 est un nombre entier négatif

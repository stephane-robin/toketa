Exercice 4 : qu'affiche le code suivant (travail sans ordinateur) ? Expliquez étape par étape ce qui produit le code.

n = 0
while n < 12:
    n = n + 1
    if n % 4 == 0:
        continue
    elif n % 7 == 0:
        break
    print(n)

Exercice 27 : créez un code qui demande à l'utilisateur son mot de passe et tant que celui-ci n'est pas le bon, le programme affiche "accès refusé". Le bon mot de passe est "qweryxc12". Lorsque le mot de passe est correct, le programme affiche "Bienvenue sur votre compte".

pw = "qweryxc12"

saisie = ""

while saisie != pw:
    saisie = input('choisissez un mot de passe : ')

    if saisie == pw:
        print("Bienvenue sur votre compte")
    else:
        print("accès refusé")

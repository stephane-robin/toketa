Effectuer des calculs avec des variables

On peut calculer la valeur d'une variable en utilisant celle d'autres variables. Par exemple si on veut trouver la hauteur d'un muret composé de pots surmontés d'arbustes, on peut effectuer le calcul suivant

hauteur_pots = 0.6
hauteur_arbustes = 1.2
hauteur_muret = hauteur_pots + hauteur_arbustes

L'avantage d'utiliser des variables plutôt que des valeurs directement est le même que celui du calcul littéral en mathématiques. Cela rend le calcul plus facile lorsqu'on doit changer les valeurs initiales.

Les principales opérations

Il est possible de faire des calculs à l'aide d'opérations qui servent à modifier la valeur des variables. Les virgules servent de séparateurs en Python et il faut utiliser un point pour séparer la valeur entière de la valeur décimale dans un nombre réel, exactement comme dans la langue anglaise. Par exemple, pour affecter la valeur 17,2 à la variable x, on écrit

x = 17.2

Pour afficher la valeur de x, il suffit maintenant d'utiliser l'instruction :

print(x)

qui affichera 17.2.

Pour les nombres entiers (int) et les nombres réels (float), on peut utiliser les opérations suivantes :

+ pour l'addition

L'instruction ci-dessous donne la valeur 8 à la variable x.

x = 5 + 3

- pour la soustraction

L'instruction ci-dessous donne la valeur 3 à la variable x.

x = 5 - 2

* pour la multiplication

L'instruction ci-dessous donne la valeur 35 à la variable x.

x = 5 * 7

/ pour la division

L'instruction ci-dessous donne la valeur 1.4 à la variable x.

x = 7 / 5

Le résultat obtenu lors d'une division est un nombre décimal.

A toi de jouer : écris un script qui calcule la vitesse moyenne d'un cycliste, connaissant la distance de montée de 16 km, la distance de descente de 51 km, le temps de montée de 1,2 heures et le temps de descente de 1,5 heures.
Indication : la vitesse moyenne se calcule en faisant la moyenne de la vitesse de montée et de la vitesse de descente. Une vitesse est le quotient d'une distance par un temps, d'où l'unité km/h.

Exécuter Python à la console

On peut tout à fait utiliser Python comme une simple calculatrice. Dans ce cas, il suffit d'écrire les lignes suivantes pour notre programme :

vitesse_moyenne = (16 + 51) / (1.2 + 1.5)
print(vitesse_moyenne)

Toutefois, il est souvent astucieux de créer plusieurs variables et de détailler nos calculs afin de pouvoir réutiliser des portions de code ou les variables dans d'autres parties du programme. On pourrait donc écrire :

distance_montee = 16 # en km
temps_montee = 1.2 # en heures
distance_descente = 51
temps_descente = 1.5

vitesse_moyenne = (distance_montee + distance_descente) / (temps_montee + temps_descente)
print(vitesse_moyenne)

** pour la puissance

L'instruction ci-dessous donne la valeur 49 à la variable x.

x = 7 ** 2

// pour le quotient d'une division

Il est également possible d'effectuer une division euclidienne. Par exemple, on sait que 7 = 1 ✕ 5 + 2, donc le quotient de cette opération vaut 1 et le reste vaut 2.

L'instruction ci-dessous donne la valeur 1 à la variable x.

x = 7 // 5

Attention la division décimale de 7 par 5 donne 1.4, mais la division euclidienne de 7 par 5 donne 1 fois 5 plus 2. Ainsi le quotient vaut 1 et le reste de division vaut 2, ce qui explique le résultat obtenu.

% pour le reste de division

L'instruction ci-dessous donne la valeur 2 à la variable x. On utilise encore une fois la division euclidienne. Celle-ci permet de mettre en valeur l'opérateur modulo % pour savoir si un nombre est multiple d'un autre, ou si une séquence de caractères peut être regroupée en blocs de 8 éléments.

x = 7 % 5

Les règles de priorité

Les parenthèses et les règles de priorité des opérations jouent le même rôle en Python qu'en mathématiques. Les opérations suivantes fournissent donc des résultats différents : (x + y ✕ z) et (x + y) ✕ z.

On pourra s'en convaincre en exécutant le code suivant :

x = 3
y = 8
z = 9

print(x + y * z) # affiche 75
print((x + y) * z) # affiche 99

Bien comprendre les changements successifs de valeurs

Voici un exemple simple d'opérations :

a = 5
b = 8
a = b + 3
b = 2

On peut récapituler les valeurs successives prises par a et b dans le tableau suivant :

instructionvaleur de avaleur de b
a = 55
b = 858
a = b + 8118
b = 2112

On remarque que lorsqu'on change la valeur de a avec l'instruction a = b + 3, b n'aucune raison de changer de valeur. De même lorsqu'on change la valeur de b avec l'instruction b = 2, a n'a aucune raison de changer de valeur. Il n'y a pas d'effect rétroactif. Les lignes sont interprétées les unes à la suite des autres.

REMARQUABLE ...

Une variable ne conserve en mémoire que la dernière valeur enregistrée. Les valeurs précédentes sont perdues.

Voici un autre exemple d'opérations :

x = 3
x = 2 * x - 2

Un mathématicien aurait tendance à vouloir résoudre l'équation x = 2x - 2 et trouverait x = 2, ce qui ne correspond pas à la valeur de x donnée au départ. Attention, il ne s'agit pas d'une égalité et il n'est jamais demandé de la résoudre en tant que telle. Il faut juste affecter de nouvelles valeurs à la variable x. Ainsi dans ce cas :

  • x prend la valeur 3

  • x prend la valeur 2 fois x moins 2, c'est à dire 2 fois 3 moins 2 donc 4

REMARQUABLE ...

Le signe égal = est utilisé pour affecter une valeur à une variable. Il n'exprime pas une équation qu'il faudrait résoudre.

Comment inverser les valeurs de deux variables

Imaginons que la variable A prenne la valeur 3 et la variable B la valeur 7. Comment faire pour que le programme inverse ces deux valeurs de manière automatisée, sans que le développeur tape manuellement B = 3 puis A = 7 ?

On peut imaginer les instructions successives suivantes :

B = A
la variable B prend la valeur de A c'est à dire 3
A = B
la variable A prend la valeur de B c'est à dire 3

On constate qu'avec cette stratégie, les deux variables prennent la valeur 3 et qu'on a perdu en mémoire la valeur 7.

Il s'agit en fait d'un problème simple mais classique en informatique. Pour comprendre la stratégie à aborder, il suffit d'imaginer deux fichiers Word appelés fichier1.docx et fichier2.docx. Si on veut changer le nom de fichier1.docx en fichier2.docx, le système d'exploitation de la machine refuse cette modification car on ne peut pas avoir deux fichiers du même nom dans un même dossier.

Si on bouge fichier2.docx dans un autre dossier, puis qu'on renomme fichier1.docx en fichier2.docx, cela va marcher mais on ne pourra plus faire revenir fichier2.docx dans son dossier initial, puisqu'on aurait dans ce cas deux fichiers du même nom.

Il faut en fait utiliser un nom de fichier temporaire pour fichier1.docx, par exemple l'appeler temp.docx. On peut ensuite renommer fichier2.docx en l'appelant fichier1.docx. Finalement, il faut renommer temp.docx en fichier2.docx.

A toi de jouer : si le principe vous échappe encore, il suffit de créer deux fichiers vides qu'on pourra appeler fichier1.docx et fichier2.docx, puis de suivre l'exemple ci-dessus en manipulant les fichiers proposés.

Tous ces fichiers peuvent également contenir des instructions, mais le but est d'apprendre à interchanger les noms des fichiers, pas de modifier ce qu'ils contiennent.

Le principe est le même pour les noms de variables. On ne peut pas avoir deux noms de variables identiques dans un même programme. Pour inverser les noms des variables, il faut donc recourir à une variable temporaire, qui permettra de ne pas perdre la valeur d'une des variables.

A toi de jouer : écrire un script qui permet d'inverser les valeurs de deux variables x et y.

A = 3
B = 7
# completer la suite ...
Exécuter Python à la console 

A = 3
B = 7
print("A = ", A)
print("B = ", B)

temp = A # on conserve la valeur initiale de A en mémoire
A = B # A prend la valeur de B
B = temp # B prend la valeur initiale de A

print("A = ", A)
print("B = ", A)

On utilise ici une variable temporaire temp afin de conserver en mémoire la valeur initiale de A. A noter qu'en Python, il existe une méthode plus simple :

A = 3
B = 7

A, B = B, A

print("A = ", A)
print("B = ", B)

Remarque : en Python, on peut définir plusieurs variables sur une même ligne, en les séparant par une virgule. Par exemple :

nom, prenom = "Gulliver", "Cedric"

Il est possible de répéter une action avec une boucle while, même si on préfère en général utiliser une boucle for dans ce cas, ce que nous étudierons plus tard. Pour cela, il faut créer un compteur qu'on fait varier, et il faut comparer ce compteur à une certaine valeur pour savoir quand s'arrêter.

A vous de jouer : testez le code suivant qui permet de compter jusqu'à 10 :

compteur = 1

while compteur < 11:
    print(compteur)
    compteur = compteur + 1

On sait maintenant compter jusqu'à 10 en utilisant une boucle while. Mais comment faire pour sauter un chiffre, par exemple pour ne pas afficher le chiffre 6 ?

A vous de jouer : essayez maintenant d'écrire un code utilisant un boucle while, qui compte jusqu'à 10 en affichant ces nombres sur une même ligne, et sans afficher 6.

compteur = 1

while compteur < 11:

    if compteur != 6:
        print(compteur)

    compteur = compteur + 1

Notons qu'on peut également utiliser l'instruction continue, qui permet de sauter un tour de boucle.

compteur = 1

while compteur < 11:

    if compteur == 6:
        compteur = compteur + 1
        continue

    print(compteur, end=" ")
    compteur = compteur + 1

Le risque avec une boucle while est que la condition d'arrêt ne soit jamais remplie, ce qui ne permet pas à la boucle de se terminer. On crée ainsi une boucle infinie qui bloque le programme. Par exemple :

compteur = 10
while compteur >= 0:
    print("le compteur est positif")

N'exécutez pas le code ci-dessus car la variable compteur sera toujours supérieure à 0 (elle vaut 10) et le programme ne se terminera jamais. Certains langages comme PHP ne permettent pas de dépasser un certain nombre de tours, mais ce n'est pas le cas de Python qui continue à travailler tant qu'on ne l'arrête pas. Il faut dans ce cas appuyer sur la touche Stop de l'environnement de travail ou Ctrl+C au terminal pour arrêter manuellement l'exécution du programme.

Python ne connaît pas de méthode isEmpty() comme d'autres langages. Il faut donc tester la longueur de la chaîne de caractères nom à la place.

On sait maintenant demander à l'utilisateur de saisir son nom, mais comment éviter qu'il se contente de saisir un espace vide ?

la fonction strip() supprime les espaces devant et derrière les caractères. Ainsi le code suivant permet de mieux visualiser si des espaces se cachent autour d'un nom, ou si un nom est composé d'espaces :

nom = " Antoine"
print(len(nom))  # affiche 8
print(len(nom.strip()))  # affiche 7

nom = ""
print(len(nom))  # affiche 0

nom = " "
print(len(nom))  # affiche 1
print(len(nom.strip()))  # affiche 0

A vous de jouer : écrire un code qui affiche Quel est votre nom ? tant que l'utilisateur ne saisit pas un nom au clavier, et qui n'accepte pas une réponse vide ou un simple espace. La valeur saisie sera stockée dans une variable nom.

Il suffit de reprendre le code de la question précédente en modifiant la condition de la boucle while comme suit :

nom = ""
while len(nom.strip()) == 0:
    nom = input("Quel est votre nom ?")

Il est parfois utile de demander à l'utilisateur de saisir un nombre entier au clavier. On a déjà vu que, par défaut, ce nombre est stocké en mémoire sous forme de chaîne de caractères. Toutefois, afin d'effectuer des calculs avec ce nombre, il est essentiel de pouvoir le transformer en entier. La fonction isnumeric() assure la compatibilité de cette conversion. On peut donc être amené à écrire une boucle while qui demande à l'utilisateur de saisir un nombre entier et recommence tant que celui-ci saisit autre chose.

A vous de jouer : afficher Quel est votre âge ? tant que l'utilisateur ne saisit pas un nombre entier au clavier. Indication : utiliser la méthode age.isnumeric()

age = ""
while age.isnumeric() == False:
    age = input("Quel est votre âge ?")

Notons que isnumeric() s'applique à une chaîne de caractères. C'est pour cette raison que la variable age est définie au départ comme étant "" et non pas None.

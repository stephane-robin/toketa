La bibliothèque turtle permet de dessiner en utilisant le langage Python. Pour ceux qui sont familiers avec le langage Scratch, les méthodes utilisées sont assez semblables. Il faut avec turtle écrire du code au lieu de déplacer des blocs Scratch.

Les principales commandes à connaître sont les suivantes :

- pour la fenêtre de travail

import turtle as tu  # il faut importer la bibliothèque turtle
t = tu.Turtle()  # on crée un objet Turtle qui ouvre une fenêtre de travail
tu.title("mon titre")  # définit le titre de la fenêtre
tu.setup()  # définit la position et les dimensions de la fenêtre
tu.bgcolor("blue")  # définit la couleur de fond de la fenêtre en bleu
tu.bgpic("monImage.png")  # définit l'image de fond de la fenêtre
tu.exitonclick()  # ferme la fenêtre en cliquant
tu.hideturtle()  # enlève le curseur de dessin
tu.showturtle()  # montre le curseur de dessin

- pour la position de l'objet dans la fenêtre, la position (0, 0) est placée au centre de la fenêtre

tu.setposition(50, 80)  # place l'objet à la position 50 pixels (abscisse), 80 pixels (ordonnée)
tu.forward(50)  # avance de 50 pixels
tu.backward(50)  # recule de 50 pixels

- pour l'orientation de l'objet, par défaut il se trouve orienté vers la droite et l'angle croît dans le sens anti-horaire (inverse des aiguilles d'une montre)

tu.setheading(30)  # oriente l'objet à 30 degrés
tu.left(30)  # oriente l'objet de 30 degrés vers la gauche de sa position
tu.right(30)  # oriente l'objet de 30 degrés vers la droite de sa position

Les principales orientations de setheading() sont 0 pour East, 90 pour North, 180 pour West et 270 pour South.

- pour tracer un dessin

tu.penup()  # lève le stylo
tu.pendown()  # baisse le stylo pour écrire
tu.pencolor("yellow")  # choisit la couleur jaune pour le stylo
tu.pensize(2)  # définit l'épaisseur des traits à 2 pixels
tu.fillcolor("red")  # définit la couleur de remplissage d'une forme en rouge
tu.begin_fill()  # commence à remplir une forme
tu.end_fill()  # s'arrête de remplir une forme
tu.circle(20)  # trace un cercle de rayon 20 pixels

Les principales couleurs sont les suivantes : black, white, grey, red, orange, green, blue, navy, yellow, gold, tan, brown, sienna, wheat, cyan, pink, salmon, violet, purple. Il est possible d'utiliser une palette entière de couleurs, qu'on retrouve par exemple sur le site trinket.

Ecrire ensuite le code de ce triangle dans une fonction qu'on appellera triangle(), qui prend comme paramètre la longueur du côté du triangle

def triangle(cote):

    tu.pendown()

    tu.forward(cote)

Etape 3

Pour dessiner la forme présentée ci-dessous, il faut répéter plusieurs 41 fois la construction du triangle dont la longueur des côtés varie de 20 à 60. A chaque tour de boucle il faut avancer de 20 pixels et s'orienter de 100 degrés vers la droite. A vous de joeur.

import turtle as tu

t = tu.Turtle()
tu.title("TP turtle")
    
def triangle(cote):
    tu.fillcolor("sienna")
    tu.begin_fill()
    tu.pencolor("navy")
    tu.pensize(2)
    tu.pendown()
    tu.forward(cote)
    tu.left(120)
    tu.forward(cote)
    tu.left(120)
    tu.forward(cote)
    tu.end_fill()
    
for i in range(20, 61):
    triangle(i)
    tu.forward(20)
    tu.right(100)
    
tu.exitonclick()

Les formes à base d'étoiles

Etape 1

Pour commencer, dessinez une étoile de 50 pixels de côté, comme celle proposée ci-dessous.

import turtle as tu

t = tu.Turtle()
tu.title("TP turtle")
    
tu.pencolor("red")
tu.pensize(3)
    
for i in range(5):
    tu.forward(50)
    tu.left(216)
    
tu.exitonclick()

La bibliothèque random est importante dans les jeux. Elle permet en effet de tirer au sort un nombre. Par exemple pour tirer au sort un nombre compris entre 1 et 12 inclus, il faut écrire :

import random as rd
   nombre = rd.randint(1, 12)

Ecrire maintenant le code de cette étoile dans une fonction qu'on appellera etoile, qui prend comme paramètre la longueur du côté de l'étoile, et choisit au hasard la couleur du dessin parmi une liste des principales couleurs. On utilisera la bibliothèque random pour choisir au hasard.

liste_couleurs = ['black', 'grey', 'red', 'orange', 'green', 'blue', 'navy',
    'yellow', 'gold', 'tan', 'brown', 'sienna', 'wheat', 'cyan',
    'pink', 'salmon', 'violet', 'purple']

def etoile(cote):
    indice = rd.randint(0, 17)
    tu.pencolor(liste_couleurs[indice])
    tu.pensize(3)

    for i in range(5):
        tu.forward(cote)
        tu.left(216)

Etape 2

Pour dessiner la forme ci-dessous, il faut dessiner plusieurs fois l'étoile, diminuer à chaque tour la longueur du côté de l'étoile, décaler légèrement la nouvelle étoile à chaque tour, et légèrement modifier l'angle de construction de l'étoile à chaque tour.

import turtle as tu
import random as rd
    
t = tu.Turtle()
tu.title("TP turtle")
    
liste_couleurs = ['black', 'grey', 'red', 'orange', 'green', 'blue', 'navy',
                    'yellow', 'gold', 'tan', 'brown', 'sienna', 'wheat', 'cyan',
                    'pink', 'salmon', 'violet', 'purple']
    
def etoile(cote):
    indice = rd.randint(0, 17)
    tu.pencolor(liste_couleurs[indice])
    tu.pensize(3)
    
    for i in range(5):
        tu.forward(cote)
        tu.left(216)
    
for i in range(25):
    etoile(50 - i)
    tu.forward(30 - i)
    tu.left(12)
    
tu.exitonclick()

Défi

Dessinez le drapeau de la suisse en utilisant la bibliothèque turtle.

import turtle as tu

t = tu.Turtle()
tu.title("drapeau suisse")
    
def rectangle(couleur, longueur_horizontale, longueur_verticale):
    tu.fillcolor(couleur)
    tu.begin_fill()
    tu.pencolor(couleur)
    tu.pendown()
    
    for i in range(4):
        tu.setheading(i * 90)  # les orientations successives seront 0, 90, 180, 270
        if i % 2 == 0:  # on trace longueur_horizontale lorsque l'indice est pair
            tu.forward(longueur_horizontale)  # l'ordre dans lequel on trace est important
        else:  # on trace longueur_verticale lorsque l'indice est impair
            tu.forward(longueur_verticale)
    
    tu.end_fill()
    
# construction du fond rouge
rectangle("red", 250, 150)
    
# construction du rectangle blanc vertical
tu.setposition(110, 25)
rectangle("white", 25, 100)
    
# construction du rectangle blanc horizontal
tu.penup()
tu.setposition(75, 65)
rectangle("white", 100, 25)
    
tu.hideturtle()
tu.exitonclick()

La bibliothèque random

Dès qu'il est question de jeu, le hasard peut intervenir à tout moment, et il est utile pour le programmeur de savoir utiliser la bibliothèque random et la bibliothèque time.

La bibliothèque random permet de tirer un nombre au hasard ou de choisir un ou plusieurs éléments au hasard dans une liste.

Pour tirer un nombre au hasard compris entre 1 et 10 :

import random
nbre = random.randint(1, 10)

Pour tirer un élément au hasard parmi une liste :

import random
semaine = ['lundi', 'mardi', 'mercredi', 'jeudi', 'vendredi']
jour = random.choices(semaine)

Pour tirer 3 éléments au hasard parmi une liste (attention chacun des éléments choisi peut se répéter) :

import random 
semaine = ['lundi', 'mardi', 'mercredi', 'jeudi', 'vendredi']
liste_choix = random.choices(semaine, k=3)

A toi de jouer : Cette fois, on ne veut pas répéter le nom des personnes tirées au sort, ni au sein d'une équipe ni entre plusieurs équipes. Que faut-il faire pour créer 3 équipes de 3 personnes au hasard à partir d'une liste de personnes ?

Solution

import random 

deck = ["Antoine", "Hervé", "Chloé", "Yvonne", "Noémie", "Pierre", "Noam", "Aylin", "Audrey"]
equipe = deck.copy()

def retirerGroupe(groupe):
    """retire groupe de la liste effectifs"""
    global equipe
    for elt in groupe:
        equipe.remove(elt)

equipe1 = random.sample(deck, k=3)
retirerGroupe(equipe1)

equipe2 = random.sample(deck, k=3)
retirerGroupe(equipe2)

equipe3 = equipe

print(equipe1)
print(equipe2)
print(equipe3)

La bibliothèque time

Pour connaître le temps écoulé en secondes depuis la création de l'informatique (1er janvier 1970) :

import time
temps = time.time()

Cela permet notamment de connaître le temps d'exécution d'un bloc d'instructions en mesurant le temps avant l'exécution puis le temps après l'exécution :

import time
debut = time.time()
i = 0
while i < 100000:
    i += 1
print(i)
fin = time.time()

duree = fin - debut

Pour mettre en pause l'exécution d'un programme pendant 10 secondes :

import time
print("Première phrase")
time.sleep(10)
print("Deuxième phrase après la pause")

Sujet du projet

Ce projet est l'occasion d'un travail en binôme. Il vous est demandé de construire un jeu de pendu simple mais fonctionnel, sans interface graphique. Vous n'utiliserez donc pas la bibliothèque turtle. Vous afficherez sur la console des tirets à la place du mot à trouver, puis les lettres correctes trouvées par le joueur aux emplacements correspondant dans ce mot. Vous afficherez également toutes les lettres proposées par le joueur. La saisie des lettres se fera directement à la console.

Le joueur dispose de 5 tentatives pour gagner. A la 6ème tentative, le joueur perd la partie.

Vous afficherez finalement à la console si le joueur a perdu ou gagné, et dans ce cas vous rajouterez son score en pourcentage.

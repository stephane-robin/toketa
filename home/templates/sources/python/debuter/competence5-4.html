Une boucle while se construit avec :

  • une condition qui renvoie True ou False

  • une indentation pour indiquer le bloc de code à répéter

  • une variable dont la valeur change pour modifier la condition au tour suivant

Dans l'exemple ci-dessus, la condition nbreVies > 0 renvoie d'abord True puisque nbreVies vaut 3, puis cette condition est testée à chaque tour jusqu'à ce que nbreVies soit égal à 0. Dans ce cas la condition nbreVies > 0 renvoie False, le programme n'effectue pas de tour supplémentaire et exécute le code placé après la boucle while (après l'indentation).

On renarque que c'est la variable nbreVies qui varie pour modifier la condition à tester.

Pseudo-code et diagramme

On rappelle que le pseudo-code consiste à écrire en langage humain les actions que réalise le code. L'exemple de code Python ci-dessus correspond au pseudo-code suivant :

nbreVies prend la valeur 3
tant que nbreVies est supérieur à 0 
{
afficher "continuer de jouer"
nbreVies perd 1 point
}
fin tant que
afficher "Fin du jeu"

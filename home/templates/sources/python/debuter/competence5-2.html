Pour parcourir une chaîne de caractères, il est nécessaire d'en connaître la taille. En effet, le programme a besoin de savoir quand s'arrêter dans son travail. Dans l'exemple ci-dessous, la chaîne est composée de 14 caractères (les espaces comptent également pour des caractères) :

chaine = "C'est cela oui"

A vous de jouer : écrire un code qui affiche séparément chaque caractère d'une chaîne les uns en dessous des autres.

chaine = "C'est cela oui"

# completer le code ...

chaine = "C'est cela oui"

for i in range(7):
    print(chaine[compteur])

Dans le code ci-dessus on a compté le nombre de caractères de chaine pour obtenir 14. Le programme parcourt donc la chaine de caractères en partant de l'indice 0, jusqu'à l'indice 13 inclus, qui correspond à l'indice du dernier caractère. En fait, il faut laisser faire ce travail à l'ordinateur en utilisant l'instruction len(chaine) qui renvoie la longueur de cette chaîne. Ainsi le code précédent peut être améliorer de la façon suivante :

chaine = "C'est cela oui"

for i in range(len(chaine)):
    print(chaine[compteur])

On sait maintenant récupérer tous les caractères d'une chaîne pour les afficher.

Mais comment faire si on ne veut pas se contenter d'afficher un nouveau mot, mais si on veut plutôt créer un nouveau mot à partir d'un ancien mot ?

On va maintenant chercher à écrire un nouveau mot à partir d'un mot en plaçant des - entre chaque lettre.

A vous de jouer : demandez à l'utilisateur de saisir un mot puis affichez ce mot avec des - entre chaque caractère.

mot = input("veuillez saisir un mot : ")
new_mot = ""

# completer le code ...

print(new_mot)

mot = input("veuillez saisir un mot : ")
new_mot = ""

for i in range(len(mot)):
    if i == 0:
        new_mot = mot[0] 
    else:
        new_mot = new_mot + "-" + mot[i]

print(new_mot)

On constate que pour construire un nouveau mot à partir d'un mot déjà existant, il faut parcourir tous les caractères existants et construire le nouveau mot par concaténation. La concaténation (rajout d'un caractère à la suite de tous les autres caractères) est incontournable lorsqu'on construit une chaîne de caractères.

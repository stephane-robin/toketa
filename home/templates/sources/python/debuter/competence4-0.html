Un caractère peut être un nombre '8', une lettre 'a' ou un terme spécial '!'.

Un texte est composé de caractères. En informatique, on utilise le terme chaîne de caractères à la place de texte car tous les caractères sont liés par la position qu'ils occupent pour former une chaîne. Python associe à une chaîne de caractères le type str pour string, qui est structuré par des numéros appelés indices représentant l'emplacement des caractères, qu'on numérote en partant de 0 et non de 1.

Par exemple, dans le mot 'Marc', le 'M' a pour indice 0, le 'a' a pour indice 1, le 'r' a pour indice 2 et le 'c' a pour indice 3.

On remarque que l'indice le plus grand d'une chaîne de caractères est sa longueur - 1. En effet, si le premier indice était 1, son dernier indice serait la longueur de la chaîne. Mais comme le premier indice vaut 0, son dernier indice est nécessairement la longueur de la chaîne - 1.

A vous de jouer : pour bien repérer la position de chaque caractère d'une chaîne grâce à son indice, copiez le code suivant qui parcourt le mot "bonjour" indice par indice, puis exécutez-le dans une console Python.

chaine = "bonjour"
for i in range(7):
    print(i, chaine[i])

Guillemets ou apostrophes ?

Les chaînes de caractères sont enregistrées dans le programme entre des guillements " ou des apostrophes ' indifféremment. Cela permet d'utiliser des balises différentes lorsque Python collabore avec un autre langage comme HTML, JavaScript, SQL, C, ... Cette situation est fréquente lorsqu'on programme des applications Web. Certains langages comme java sont bien moins tolérants et différencient l'usage des apostrophes et des guillemets. Chance ! Python nous offre une certaine flexibilité dans le code.

Attention toutefois, lorsqu'on utilise des apostrophes ' et que la chaîne contient une apostrophe, Python pourrait l'interpréter comme la fin de la chaine. Pour éviter cela, on utilise un antislash \ qui indique à Python que le caractère suivant appartient à la chaîne. Par exemple, les codes ci-dessous sont identiques :

phrase = "Les pâtes viennent d'Italie"
phrase = 'Les pâtes viennent d\'Italie'

Le problème des accents

Lorsqu'on utilise des accents dans un texte comme par exemple en français é, à, è, ù, etc... cela crée des difficultés d'affichage pour Python qui a été construit en anglais. Il faut dans ce cas rajouter dès la première ligne de code l'instruction suivante indiquant à Python que les accents et caractères spéciaux sont autorisés et qu'il pourra les trouver dans la table utf-8 :

# -*- coding: utf-8 -*-

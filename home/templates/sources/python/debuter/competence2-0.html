Pour afficher la valeur d'une variable, il suffit d'utiliser l'instruction print avec le nom de cette variable :

print(age)

affichera 18.

On a déjà exécuté l'instruction suivante pour tester Python :

print("Hello world")

Attention, les guillemets "" sont importants dans le script ci-dessus. Essaie le même script sans les "". L'interpréteur Python pense dans ce cas que Hello est une variable dont il ne connaît pas la valeur. Les guillemets "" permettent en fait d'afficher du texte et non pas la valeur d'une variable.

A toi de jouer : pour comprendre l'utilité des guillemets "", exécute le code suivant qui différencie "prenom" et prenom :

prenom = "Antoine"
print("prenom")
print(prenom)
Exécuter Python à la console

On utilise donc les guillemets pour afficher du texte qui se trouve à l'intérieur de ces guillements, alors que sans guillemet le code affiche la valeur de la variable prenom.

Pour afficher les valeurs de plusieurs variables successives, il suffit de les séparer par le signe + dans l'instruction print

A toi de jouer : affecte ton prénom à la variable prenom, puis ton nom à la variable nom, comme dans l'exemple ci-dessous :

prenom = "Antoine"
nom = "Legendre"

Affiche ensuite à l'écran ton nom suivi de ton prénom en utilisant les deux variables que tu viens de créer.

Exécuter Python à la console 

prenom = "Antoine"
nom = "Legendre"
print(nom + prenom)

Affichage de variables dans un texte

Nous allons voir comment afficher les valeurs de plusieurs variables, ou la valeur d'une variable au sein d'un texte.

nom = "Parker"
prenom = "Peter"
print(prenom + nom) # affiche PeterParker
print(prenom + " - " + nom) # affiche Peter - Parker
print(prenom + "\t" + nom) # affiche Peter   Parker séparés par une tabulation
print(prenom + "\n" + nom) # affiche Peter Parker l'un en dessous de l'autre

On remarque donc que l'instruction print() saute automatiquement la ligne, une fois le travail effectué. Elle correspond en fait au println() qu'on retrouve dans d'autres langages. Pour forcer l'écriture de texte sur la même ligne, il faut rajouter le mot-clé end="", comme dans l'exemple ci-dessous :

nom = "Parker"
prenom = "Peter"
print(prenom, end="-")
print(nom)

Ce code affiche Peter-Parker sur la même ligne, comme souhaité. Notons l'utilisation du séparateur "-" entre Peter et Parker qu'on peut modifier à souhait.

Affichage respectant un format

On a vu ensemble qu'on pouvait afficher la valeur d'une variable au milieu d'un texte, en construisant un texte qui contient cette variable. Par exemple le code suivant :

prenom = "Antoine"
date_naissance = "17-02-2008
print(prenom + " est né le " + date_naissance)

affichera le résultat suivant :

Antoine est né le 17-02-2008

Il existe une façon plus efficace d'afficher le même résultat en utilisant l'instruction print(f) :

prenom = "Antoine"
date_naissance = "17-02-2008
print(f"{prenom} est né le {date_naissance}")

print(f"Demain nous serons {nombre_invites} invités au restaurant {nom_restaurant}")

Affichage d'une variable de type différent de str

Imaginons qu'on veuille afficher le score d'un joueur à la fin d'une partie. En imaginant qu'il ait obtenu le score de 147 et qu'il s'appelle "Bastien", on serait tempté d'écrire :

prenom = "Bastien"
score = 147

print("Le joueur " + prenom + " a obtenu le score de " + score)

Si tu exécutes ce code, tu te rendras vite compte qu'il déclenche une erreur, car prenom est une chaine de caractères et s'intègre donc parfaitement au reste du texte, mais score est un entier et ne sait pas comment faire pour compléter un texte. Il faut donc le transformer en chaîne de caractères avec l'instruction str et modifier notre code comme ceci :

prenom = "Bastien"
score = 147

print("Le joeur " + prenom + " a obtenu le score de " + str(score))

Il est également possible d'utiliser l'affichage formaté qui est tout à fait adapté à l'affichage de nombres :

prenom = "Bastien"
score = 147

print(f"Le joueur {prenom} a obtenu le score de {score})

Affichage respectant la structure donnée

Il est possible d'afficher un texte en respectant les sauts de ligne et les espaces comme dans le texte d'origine. Pour cela, il convient d'utiliser """, comme dans l'exemple ci-dessous :

print("""Deux coeurs qui s'aiment,
n'allez pas chercher plus loin la poésie;
et deux baisers qui dialoguent,
n'allez pas chercher plus loin la musique. (V. Hugo)""")

qui va afficher :

Deux coeurs qui s'aiment,
n'allez pas chercher plus loin la poésie;
et deux baisers qui dialoguent,
n'allez pas chercher plus loin la musique. (V. Hugo)

Il est possible de couper les lignes de codes Python afin d'afficher toutes les instructions sur une même page sans devoir bouger la souris pour lire le code, en utilisant un antislash \ qui indique à Python que le code continue. En Effet, contrairement à d'autres langages où un point-virgule ; symbolise la fin d'une instruction, en Python le saut de ligne ferme une instruction. Voici quelques exemples d'utilisation de l'antislash :

print("Deux coeurs qui s'aiment,\
n'allez pas chercher plus loin la poésie;\
et deux baisers qui dialoguent,\
n'allez pas chercher plus loin la musique. (V. Hugo)")

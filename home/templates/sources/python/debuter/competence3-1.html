L'indentation

L'indentation est le fait de décaler le code vers la droite (d'une tabulation ou de 4 espaces). C'est essentiel en Python qui n'utilise par d'accolades et choisit de commencer un bloc de code par une indentation et de rester dans ce bloc tant que le code est indenté. Par exemple dans le code suivant :

nom = "Alice"
if nom == "Alice":
    print("Bonjour Alice.")
    print("Comment allez-vous aujourd'hui ?")
print("Aujourd'hui c'est journée portes ouvertes.")

Bonjour Alice. Comment allez-vous aujourd'hui ? ne s'affichera que si le nom vaut "Alice". Les deux phrases font partie du même bloc d'instructions. En revanche, Aujourd'hui c'est journée portes ouvertes. s'affichera tout le temps, même si nom ne vaut pas "Alice", car cette instruction est indépendante de l'instruction if nom == "Alice":.

A vous de jouer : exécuter le code ci-dessus pour voir ce qui s'affiche, puis pour bien comprendre le rôle de l'indentation :

  • changez le nom 'Alice' en 'Bob' et exécutez de nouveau le code

  • indentez la dernière ligne avec la touche de tabulation avant d'exécuter le code pour constater la différence d'affichage.

Bien comprendre la structure d'une instruction if

Exécutez le code suivant pour bien comprendre le cheminement suivi par le programme.

1 | cheveux = "bruns"
2 | yeux = "bleus"
3 |
4 | if cheveux == "bruns":
5 |    print("May a les cheveux bruns")
6 | else:
7 |    if yeux == "bleus":
8 |        print("May a les yeux bleus")
9 |
10| print("Voilà c'est tout")

Si les cheveux sont bruns, le programme exécute la ligne 4 mais n'entre jamais dans le bloc de la ligne 6. Il reprend ensuite à la ligne 10 pour afficher Voilà c'est tout. Du coup on ne saura jamais si les yeux de May sont bleus.

A noter qu'en Python, l'instruction else: if peut se condenser en une instruction elif. On peut donc ré-écrire le code précédent de la façon suivante :

1 | cheveux = "bruns"
2 | yeux = "bleus"
3 |
4 | if cheveux == "bruns":
5 |    print("May a les cheveux bruns")
6 | elif yeux == "bleus":
7 |    print("May a les yeux bleus")
8 |
9 | print("Voilà c'est tout")

Mais pourquoi utiliser l'instruction elif plutôt qu'une instruction if plus simple ? L'instruction elif est exclusive, c'est à dire que le programme testera si les yeux sont bleus uniquement si les cheveux ne sont pas bruns. Dans le cas d'une instruction if, le programme testera si les yeux sont bleus dans tous les cas, cheveux bruns ou pas.

A vous de jouer : pour mieux comprendre, exécutez le code suivant dans lequel tous les blocs sont systématiquement exécutés :

1 | cheveux = "bruns"
2 | yeux = "bleus"
3 |
4 | if cheveux == "bruns":
5 |    print("May a les cheveux bruns")
6 |
7 | if yeux == "bleus":
8 |    print("May a les yeux bleus")
9 |
10| print("Voilà c'est tout")

Si les cheveux sont bruns, le programme exécute la ligne 4, puis si les yeux sont bleus, le programme exécute la ligne 8. Il reprend ensuite à la ligne 10 pour afficher Voilà c'est tout.

Python permet de répéter une instruction un certain nombre de fois. Lorsqu'on connaît le nombre de répétitions, on utilise la boucle for. Lorsqu'on ne connaît pas le nombre de répétitions et que celui-ci dépend de l'évolution de la situation (par exemple si le nombre de vies d'un joueur devient nul), alors dans ce cas on utilise la boucle while.

Pour répéter bonjour 5 fois, il suffit simplement d'écrire

for i in range(5):
    print("bonjour")

A noter encore une fois l'indentation du code devant l'instruction print("bonjour") pour indiquer que cette instruction fait partie du bloc constitué par la boucle for.

Si tu as déjà pratiqué la programmation avec Scratch, le script ci-dessus correspond en fait au code Scratch suivant :

Dans l'instruction for i in range(5):, l'indice i va prendre les valeurs entières comprises entre 0 et 4. Il y aura de ce fait 5 répétitions de l'instruction print("bonjour"). En réalité, l'instruction range(5) crée une liste de 5 nombres allant de 0 à 4 dans laquelle va piocher l'indice i. Mais tu n'as pas vraiment besoin de le savoir pour utiliser la boucle for.

REMARQUABLE ...

L'indice i d'une boucle for i in range(4) commence par défaut à 0. La dernière valeur prise par cet indice est l'entier strictement inférieur à 4.

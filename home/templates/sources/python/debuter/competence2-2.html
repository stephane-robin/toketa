Exercice 7 : écrire un programme qui demande un nombre à l’utilisateur, puis affiche si le terme saisi par l'utilisateur peut être transformé en entier (l'affichage sera uniquement True ou False, indication, pensez à isnumeric()). Finalement, le code affiche "[nombre] fois 4 est égal à " (où [nombre] représente la valeur du nombre saisi), suivi du résultat de cette multiplication. Par exemple si la valeur saisie est 3, le code affichera "True" puis "3 fois 4 est égal à 12".

nombre = input("veuillez saisir un nombre : ")
print(nombre.isnumeric())

# on transforme la valeur saisie en entier afin d'effectuer le calcul
nombre = int(nombre)
print(nombre + " fois 4 est égal à " + nombre * 4)

Nous pouvons maintenant compliquer un petit peu les choses. Un palindrome est un mot qui se lit de la même façon à l'endroit qu'à l'envers. Ainsi par exemple kayak est un palindrome alors que bonjour n'en est pas un.

Nous avons vu comment inverser un mot saisi par l'utilisateur. Pour tester si un mot est un palindrome, il suffit de comparer le mot inversé avec le mot de départ pour répondre à la question.

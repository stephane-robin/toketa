L'objectif de ce travail pratique est d'écrire 4 fonctions Python permettant de mettre à jour nos connaissances.

  • une fonction decimalToBinaire() qui transforme l'écriture décimale d'un nombre en écriture binaire,

Proposition de correction pour decimalToBinaire()

Le code suivant permet de convertir un nombre en base 10 dans son écriture binaire. Il ne s'agit pas du code le plus court ou le plus performant, mais il permet de comprendre comment procéder pour obtenir le résultat.

def decimalToBinaire(decimal):
    binaire = ''

    while decimal > 0:
        # decimal%2 doit etre transformé en string avant d'être concaténé
        binaire = str(decimal % 2) + binaire
        decimal = decimal // 2

    return binaire

Le code ci-dessus fournit le résultat 110100 lorsqu'on exécute

print(decimalToBinaire(52))

A noter l'existence de la fonction bin() qui fournit le même résultat : 

def decimalToBinaire2(n):
    return bin(n).replace("0b", "")

A noter dans ce cas que le résultat de la fonction bin() est perçu comme un nombre binaire par l'interpréteur Python, et qu'il faut donc remplacer le "0b" au début de ce nombre par un caractère vide.

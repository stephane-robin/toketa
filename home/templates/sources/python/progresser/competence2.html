A quoi sert la structure de dictionnaire ?

Cela peut servir à structurer les données de façon plus intuitive ou plus simple, en utilisant un système clé-valeur. L'inconvénient d'une liste, c'est que ses éléments sont ordonnés, et que si on veut faire appel à un élément en particulier, il faut connaître son indice.

Dans le cas d'un dictionnaire, on ne peut pas ordonner ses éléments, puisqu'ils ne disposent d'aucun indice. Il suffit de connaître la clé pour obtenir la valeur correspondante. C'est un peu comme un ensemble de boîtes toutes identiques dans un sac. Si on récupère une boìte sur laquelle se trouve un nom (sa clé), on peut l'ouvrir et obtenir la valeur qui se trouve à l'intérieur. On ne peut donc pas avoir 2 clés identiques, ou le programme ne saurait pas quelle valeur choisir.

La structure d'un dictionnaire est proche de celle d'un fichier JSON, ce qui rend les interactions entre les 2 formats de données très faciles. Voici un exemple de dictionnaire :

dico = {
    'prenom': 'Karl',
    'classe': '1B2',
    'age': 17
}

Manipuler les dictionnaires

Considérons le dictionnaire suivant :

langages = {
    'python': 'très populaire',
    'javascript': 'langage web',
    'java': 'langage objet'
}

Pour obtenir 'très populaire', il faut utiliser l'instruction :

langages['python']

Pour tester si 'python' se trouve dans le dictionnaire, il faut utiliser l'instruction :

print('python' in langages)

Attention, seules les clés peuvent être atteintes de cette façon.

Les instructions langages.keys() et langages.values() récupèrent respectivement toutes les clés et toutes les valeurs du dictionnaire langages.

Pour afficher un dictionnaire complet, on peut utiliser :

for elt in langages:
    print(elt + ": " + langages[elt])

ou 

for cle, valeur in langages.items():
    print(cle + ": " + valeur)

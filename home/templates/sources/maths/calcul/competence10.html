Vous cherchez à écrire des expressions mathématiques avec un rendu de qualité, ou à créer des évaluations pour vos élèves en utilisant le même format que votre groupe de travail, ou à conserver vos travaux dans un format libre de droits et pérenne dans le temps. LaTeX est la solution pour vous.

Seulement voilà, vous ne disposez pas de beaucoup de temps pour apprendre à utiliser toutes les fonctionnalités LaTeX. Vous voulez aller droit au but et pouvoir produire des résultats rapidement. Nous allons vous proposer des solutions rapides et prêtes à l'emploi.

Les principales difficultés que rencontre toute personne qui se lance en LaTeX sont les suivantes :

  • on ne voit pas immédiatement le rendu de ce qu'on écrit
  • les lignes de codes nécessaires sont difficiles à interprétées pour l'oeil humain
  • il est nécessaire de compiler le programme et donc de débugger celui-ci
  • il est fréquent que des packages ne soient pas compatibles
  • l'installation de LaTeX et des différents package peut devenir un cauchemar
  • la gestion de la mise en page, notamment des images, des espaces, des césures, etc... fait perdre beaucoup de temps

Ces difficultés engendrent un surcoût de travail et une perte de temps non négligeable. Ainsi la courbe de l'apprentissage avant de profiter des avantages de LaTeX n'est pas très favorable et conduit beaucoup de personnes à abandonner de mode d'édition de texte.

Voyons maintenant les avantages de LaTeX, car il en existe tout de même :

  • une fois que l'oeil s'est familiarisé avec les mots-clés LaTeX, et les pages de code, le temps passé à écrire le document est optimisé
  • le rendu obtenu, notamment pour les formules mathématique est très professionnel
  • on peut intégrer du LaTeX dans d'autres applications (Geogebra, Plickers, sites Web, document Word,...)
  • lorsqu'on écrit un livre de plus de 80 pages, LaTeX ne risque pas de planter lors de l'édition, contrairement à Word ou LibreOffice
  • s'agissant d'un format libre de droits, on pourra toujours réutiliser son travail dans le futur sans le risque qu'une nouvelle version commercial rende caduque tous les documents produits

Ce dernier avantage est la raison pour laquelle je recommande LaTeX. Certes, il existe des alternatives telles que le markdown mk ou encore les fichiers LibreOffice odt, mais la syntaxe utilisée n'est pas vraiment plus simple et on perd beaucoup de temps à rédiger contrairement à LaTeX.

Attention toutefois, LaTeX n'est pas un langage de programmation à part entière, même s'il a besoin de compiler pour se transformer en format pdf. Il s'agit d'un langage de communication. LaTeX est adapté à la création de livres mais sa courbe d'apprentissage le rend inutile pour toute production qui ne comprend pas de formule. J'entends pas là que vouloir écrire un livre d'histoire en LaTeX est une perte de temps et une source inutile de difficulté. Il est dans ce cas bien préférable d'utiliser Word ou LibreOffice. En revanche si votre travail intègre du contenu mathématique, LaTeX est fait pour vous.

Un modèle pour les évaluations

Overleaf

Voyons maintenant comment optimiser notre courbe d'apprentissage du LaTeX. Tout d'abord, il faut éviter une installation manuelle sur un poste afin d'éviter la gestion des packages et des chemin d'accès. Pour cela, je recommande l'utilisation d'un éditeur/compilateur en ligne. Inscrivez-vous par exemple sur Overleaf à l'adresse suivante www.overleaf.comet ouvrez votre premier projet.

Un projet correspond tout simplement à un fichier LaTeX.

Vous allez cliquer sur 'Nouveau projet', puis 'Projet vide'. Appelez-le par exemple 'modeleEvaluation.tex'. L'extension .TEX est le format des documents LaTeX avant qu'ils soient compilés pour produire un document .PDF.

Utilisation d'une classe déjà définie

Vous allez maintenant remplacer le code pr-rempli par une structure minimaliste que vous pourrez ensuite compléter à souhait.

% DOCUMENT A COMPILER EN PDFLATEX

% definit le type de document
\documentclass[twoside,a4paper,11pt,frenchb]{evaluation}

\title{Evaluation}
% pour ne pas afficher la date
\date{}
% pour ne pas afficher l'auteur
\author{}

\begin{document}

% pour afficher titre en date en fonction des choix effectues
\maketitle

NOM : \\

Note :    \hspace{3cm}      / 20

\vspace{1cm}

Répondre sur une feuille séparée. Calculatrice autorisée. Prêt de calculatrice interdit durant l'évaluation.

% les sections ne sont pas numerotes grace aux *
\section*{Soin (1 point)}

\section*{Exercice 1 - 3 points}

A compléter...

\section*{Exercice 2 - 3 points}

A compléter ...

\section*{Exercice 3 - 3 points}

A compléter...

\end{document}

Le type de ce document avec les différentes bibliothèques dont il dépend est défini dans les premières lignes de code. Il s'agit d'une classe 'evaluation'.

Vous aurez également besoin de cette classe 'evaluation'. Vous pourrez la télécharger dans le dépôt Git à l'adresse suivante : evaluation.cls

Si vous ouvrez le fichier 'evaluation.cls' (l'extension .CLS correspondant au format d'une classe LaTeX), vous obtenez le code suivant :

\NeedsTeXFormat{LaTeX2e}[2023-06-30]
\ProvidesClass{evaluation}[2023-06-30 v1 classe pour evaluation]

\LoadClassWithOptions{article}

\author{Stephane Robin}

% pour ecrire les accents
\RequirePackage[utf8]{inputenc}
% pour la mise en page du document
\RequirePackage[top=3.5cm,bottom=3.0cm,left=2.5cm,right=2.5cm,dvips]{geometry} 
% pour enlever l'indentation de la 1ere ligne d'un paragraphe
\RequirePackage{parskip}
% pour la coloration des codes Python
\RequirePackage{minted}
% pour les symboles mathematiques
\RequirePackage{amssymb}
\RequirePackage{amsmath}
\RequirePackage{amsthm}
\RequirePackage{mathrsfs} 
% pour les caracteres grecs
\RequirePackage{bm}
% pour la coloration de texte
\RequirePackage{color}
% pour les tableaux
\RequirePackage{array}
% pour inserer des images
\RequirePackage{caption}  
% pour les legendes des images
\RequirePackage[font=footnotesize,labelfont=bf]{caption} 

\endinput

Une fois que vous aurez téléchargé cette classe, placez-là dans le même dossier que votre fichier 'modeleEvaluation.tex', et le tour est joué. Vous disposez maintenant d'un modèle d'évaluation préfabriqué. Il ne vous reste plus qu'à adapter le contenu en fonction de vos besoins à l'aide des instructions standard LaTeX.

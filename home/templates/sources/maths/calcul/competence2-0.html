Introduction

On sait que \(3^2 = 3 \times 3 = 9\), donc si on regarde le chiffre 9, on peut en déduire que le chiffre qui manque dans l'expression suivante \(... \times ... = 9\) est 3.

3 est appelé la racine carrée de 9.

On peut maintenant facilement trouver la racine carrée de 16. Il s'agit de 4 puisque \(4 \times 4=16\).

Mais qu'en est-il de la racine carrée de 18 ? Il faut pour cela utiliser la calculatrice qui nous donne comme résultat 4,24.

DEFINITION ...

La racine carrée de \(a\) est le nombre positif noté \(\sqrt{a}\) tel que \((\sqrt{a})^2=a\).

Dans cette définition, le nombre \(a\) et sa racine carrée \(\sqrt{a}\) sont toujours positifs.

Le symbole \(\sqrt{}\) est appelé radical.

Il est tout à fait normal qu'une racine carrée soit toujours positive, puisque lorsqu'on multiplie un nombre par lui-même, d'après la règle des signes, on obtient un signe +.

A noter que \(\sqrt{0}=0\) et \(\sqrt{1}=1\).

A vous de jouer : utiliser votre calculatrice pour trouver les valeurs de \(\sqrt{32}\), \(\sqrt{3}\), \(\sqrt{175}\), \(\sqrt{-12}\).

Solution :

\(\sqrt{32}\)=5,66, \(\sqrt{3}\)=1,73, \(\sqrt{175}\)=13,23, \(\sqrt{-12}\) n'existe pas (n'admet pas de valeur réelle).

Les règles de calcul

Il faut savoir appliquer les règles suivantes afin de pouvoir calculer dès qu'une expression contient des racines carrées :

A RETENIR ...

\(\sqrt{a \times b}=\sqrt{a}\times\sqrt{b}\) avec \(a, b > 0\)

\(\sqrt{\frac{a}{b}}=\frac{\sqrt{a}}{\sqrt{b}}\) avec \(a, b > 0\)

\(\sqrt{a^n}=(\sqrt{a})^n\) avec \(a > 0\)

En utilisant la dernière règle on peut par exemple écrire que \((\sqrt{2})^6=\sqrt{2^6}=\sqrt{64}=8\)

Exercice type : Transformer \(\sqrt{45}\) en un produit de deux racines carrées.

Solution : \(\sqrt{45}=\sqrt{15 \times 3}=\sqrt{15}\times\sqrt{3}\).

A vous de jouer : Simplifier \(\sqrt{48}\)

Solution :

\(\sqrt{48}=\sqrt{6\times 8}=\sqrt{6}\times\sqrt{8}\)

Exercice type : simplifier \(\frac{\sqrt{24}}{\sqrt{3}}\)

Solution : il n'est pas facile d'effectuer des calculs avec \(\sqrt{24}\) et \(\sqrt{3}\). Il est possible d'utiliser la 2ème règle de calcul pour obtenir \(\frac{\sqrt{24}}{\sqrt{3}}=\sqrt{\frac{24}{3}}=\sqrt{8}\). Nous nous retrouvons maintenant avec une seule racine au lieu de deux racines, ce qui est un peu plus simple à utiliser.

A vous de jouer : simplifier \(\frac{\sqrt{45}}{\sqrt{5}}\)

Solution :

\(\frac{\sqrt{45}}{\sqrt{5}}=\sqrt{9}=3\)

⚡ Attention au piège. Il n'existe pas de règle de calcul simple pour \(\sqrt{a+b}\). Utilisez votre calculatrice pour trouver la valeur de \(\sqrt{38}\), puis la valeur de \(\sqrt{21}+\sqrt{17}\). Vous constatez qu'elles sont différentes. Ainsi, en général \(\sqrt{a+b}\neq\sqrt{a}+\sqrt{b}\).

Bravo, vous savez maintenant utiliser les principales règles concernant les racines carrées.

Comment simplifier les racines carrées

La règle de calcul la plus importante à retenir est la suivante :

A RETENIR ...

\(\sqrt{a^2}=(\sqrt{a})^2\) avec \(a > 0\)

Elle permet de simplifier l'écriture des racines carrées, une fois qu'on a fait ressortir un carré parfait.

Cette stratégie est la plus couramment utilisée en présence de racines carrées. En effet, comme les racines carrées ne sont pas faciles à manipuler, on cherche à les transformer en nombres entiers en faisant ressortir des carrées parfaits.

A vous de jouer : faites travailler votre mémoire en énonçant le plus grand nombre de carrés parfaits que vous connaissez.

Solution : 4, 9, 16, 25, 36, 49, 64, 81, 100, 121, 144, 169, ...

Cherchons maintenant à simplifier les écritures avec radical.

Exercice type : Simplifier \(\sqrt{150}\)

Solution : il faut commencer par décomposer le nombre 150 en produit de facteurs premiers : \(150=2 \times 3 \times 5^2\), puis chercher si on trouve un carré parfait, afin de le simplifier en utilisant la dernière règle \(\sqrt{a^2}=(\sqrt{a})^2\).
Dans ce cas \(\sqrt{150}=\sqrt{2 \times 3 \times 5^2}=\sqrt{2 \times 3}\times\sqrt{5^2}=\sqrt{2\times 3}\times 5=5\sqrt{6}\)

A vous de jouer : transformer \(\sqrt{108}\) en un produit de racines carrées

Solution :

\(\sqrt{108}=\sqrt{3^3\times 2^2}=\sqrt{3}\sqrt{3^2}\sqrt{2^2}=3\times 2\sqrt{3}=6\sqrt{3}\)

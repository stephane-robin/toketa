Utiliser Geogebra pour résoudre une équation

Geogebra est un logiciel permettant de dessiner des formes géométriques avec précision et représente une aide précieuse à tout élève qui souhaite s'investir dans la géométrie. Mais savez-vous qu'il est égalememnt possible de factoriser à l'aide de Geogebra ? Nous allons découvrir ensemble comment procéder.

Geogebra permet de dessiner les courbes représentatives des fonctions, mais également de retrouver graphiquement la solution d'une équation.

1ère partie - calcul manuel

Représentation graphique

Vous allez dessiner sur votre cahier les représentations graphiques des fonctions suivantes :

  • \(f(x)=3x+1\)
  • \(g(x)=x^2-4\)

Résolution d'équations

Résoudre sur votre cahier les équations suivantes :

  • \(f(x)=0\)
  • \(g(x)=0\)

2ème partie - calcul avec Geogebra

Cela n'est pas si facile d'obtenir un résultat juste rapidement en procédant à la main. Voyons comment vérifier ces résultats quasi-instantanément.

Tout d'abord, il faut vous rendre sur le site www.geogebra.org ou télécharger l'application directement sur votre ordinateur. Nous utiliserons ensemble la version en ligne.

Puis il est préférable d'utiliser le module Geogebra classique et de définir les fonctions dans le champ en haut à gauche de la page.

Nous verrons ensemble comment tracer des courbes et résoudre une équation en lisant la valeur de x pour laquelle la courbe représentative touche l'axe des abscisses.

Utiliser Python pour résoudre une équation

Vous avez appris à résoudre des équations. C'est parfois difficile, mais c'est nécessaire d'en connaître la méthode. Le langage Python permet également de trouver la solution de certaines équations, et cela à moindre effort. Une fois que vous saurez utiliser Python correctement, cet outil performant vous permettra par exemple de vérifier le résultat de vos calculs.

Partie 1 - calcul manuel

Dans un premier temps, soyez courageux et effectuez les calculs suivants à la main, afin de comparer plus tard vos résultats et ceux de l'ordinateur.

\(2x-3 = 6x+7\)

\(23-56x = 12x-4\)

\(6x+45 = 7x+8\)

Partie 2 - Python

Nous allons maintenant retrouver les résultats précédents en utilisant des codes simples du langage Python.

En Python la résolution d'équations nécessite l'importation d'un module appelé sympy. Pour cela, commencez votre code par

from sympy import *

On déclare ensuite la variable \(x\) en écrivant

x = symbols('x')

Pour résoudre l'équation \(x-3=0\) en Python, il faut exécuter le code suivant :

print(solveset(x-3, x))

Pour résoudre l'équation \(2x+1=3x-5\) en Python il faut exécuter le code suivant :

print(solveset(2*x+1-(3*x-5), x))

Essayez maintenant d'obtenir à l'aide de Python les résultats des calculs que vous venez de faire manuellement. Pour rappel, la multiplication utilise le symbole * et la division le symbole / en Python.

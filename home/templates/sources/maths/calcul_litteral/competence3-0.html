Développer une expression en distribuant

Vous avez appris à factoriser une expression mathématique. Vous avez fait le plus difficile. Bravo...

Développer une expression mathématique, c'est faire l'opération inverse. Mais quel est l'intérêt d'obtenir une expression plus longue alors qu'on la connaît déjà sous forme de produit compact ? Il est parfois utile de développer tous les termes lorsqu'on peut les combiner avec une autre expression, pour les éliminer. C'est donc un outil important qu'il faut connaître pour manipuler les calculs.

Première situation

Exercice type : développez \(3(x+5)\)

Le chiffre 3 doit multiplier chacun des deux éléments

A vous de jouer : développez

  • \(7(x+5)=...\)
  • \(7(x-5)=...\)
  • \(5(2x+3)=...\)

Deuxième situation

Exercice type : développez \((x+2)(x+5)\)

Le chiffre 3 doit multiplier chacun des deux éléments

A vous de jouer : développez

  • \((x+5)(x+3)=...\)
  • \((2x+5)(3x+1)=...\)

On complique un peu en introduisant des signes -. Il suffit de respecter la règle des signes :

\((+\times +) \rightarrow +\)

\((+\times -) \rightarrow -\)

\((-\times +) \rightarrow -\)

\((-\times -) \rightarrow +\)

Exercice type : développez \((x+2)(x-5)\)

Le chiffre 3 doit multiplier chacun des deux éléments

A vous de jouer : développez

  • \((x+7)(x-5)=...\)
  • \(5(2x+3)(x-3)=...\)

Développer en utilisant une identité remarquable

Lorsqu'on reconnaît une identité remarquable, il est plus rapide d'utiliser la formule apprise que de développer terme à terme, même si cela reste juste.

A RETENIR ...

\((a+b)^2 = a^2+2ab+b^2\)

\((a-b)^2 = a^2-2ab+b^2\)

\((a-b)(a+b) = a^2-b^2\)

1ère identité remarquable

Exercice type : développer \((x+5)^2\)

On reconnaît la 1ère identité remarquable avec \(a=x\) et \(b=5\). On obtient donc \((x+5)^2=x^2+10x+25\).

A vous de jouer : développer

  • \((x+7)^2=...\)
  • \((3x+4)^2=...\)

2ème identité remarquable

Exercice type : développer \((x-4)^2\)

On reconnaît la 2ème identité remarquable avec \(a=x\) et \(b=4\). On obtient donc \((x-4)^2=x^2-8x+16\). Attention, il n'y a QU'UN SEUL signe - dans la formule...

A vous de jouer : développer

  • \((x-9)^2=...\)
  • \((2x-7)^2=...\)

3ème identité remarquable

Exercice type : développer \((5x-3)(5x+3)\)

On reconnaît la 3ème identité remarquable avec \(a=5x\) et \(b=3\). On obtient donc \((5x-3)(5x+3)=(5x)^2-9=25x^2-9\). Attention à bien mettre le 5 et le x au carré.

A vous de jouer : développer

  • \((4x-2)(4x+2)=...\)
  • \((3x+7)(3x-7)=...\)

Entrainement

Exercice : développez

  • \((2x-5)(x+3)=...\)
  • \((x-7)(x-5)=...\)
  • \((21(3x-x^2))=...\)
  • \(5(2x-3)(3-x)=...\)
  • \((2x-5)(2x+5)=...\)
  • \((2x-7)(2x+3)=...\)
  • \((2x-5)(2x-5)=...\)
  • \(5(2x+3)^2=...\)

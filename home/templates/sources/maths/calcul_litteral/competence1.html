Utiliser Geogebra pour factoriser

Geogebra est un logiciel permettant de dessiner des formes géométriques avec précision et représente une aide précieuse à tout élève qui souhaite s'investir dans la géométrie. Mais savez-vous qu'il est égalememnt possible de factoriser à l'aide de Geogebra ? Nous allons découvrir ensemble comment procéder.

1ère partie - calcul manuel

Vous allez factoriser les expressions suivantes sur votre cahier :

\(9x^2-42x+49=...\)

\(9-121x^2=...\)

\((x-1)(x+7)-(x-1)(4x-3)\)=...

\((x+1)(x+27)+(x+1)(2x-3)=...\)

Maintenant, passons à l'opération inverse. Vous allez développer les expressions suivantes sur votre cahier :

\((x-1)(5x+3)=...\)

\((2x-9)^2=...\)

2ème partie - calcul avec Geogebra

Cela n'est pas si facile d'obtenir un résultat juste rapidement en procédant à la main. Voyons comment vérifier ces résultats quasi-instantanément.

Tout d'abord, il faut vous rendre sur le site www.geogebra.org ou télécharger l'application directement sur votre ordinateur. Nous utiliserons ensemble la version en ligne.

Puis il faut utiliser le module calcul formel dans Geogebra Classique. Vous trouverez le reste facilement en testant le logiciel.

Utiliser Python pour factoriser

Vous avez appris à factoriser et développer des expressions mathématiques. C'est parfois difficile mais cela vous permet de résoudre certains problèmes dont vous ne pouviez trouver la solution autrement. Le langage Python permet d'obtenir un résultat similaire à moindre effort si on sait l'utiliser correctement. Nous avons vu ensemble comment utiliser GeoGebra dans ce but, voyons maintenant comment utiliser Python.

Partie 1 - calcul manuel

Dans un premier temps, soyez courageux et effectuez les calculs suivants à la main, afin de comparer plus tard vos résultats et ceux de l'ordinateur.

Partie 2 - Python

Pour exécuter les codes ci-dessous, vous pouvez appuyer sur le bouton Run de l'interpréteur Python.

Nous allons maintenant retrouver les résultats précédents en utilisant des codes simples du langage Python.

En Python, le calcul littéral nécessite l'importation d'un module appelé sympy, puis l'affectation de symboles formels x, y, z, ... aux différentes variables x, y, z, ....

Pour cela, commencez votre code par

from sympy import *
x, y, z = symbols('x, y, z')

Factorisation

Pour factoriser une expression mathématique E, sympy utilise l'instruction factor(E).

Par exemple, on sait que \((x-1)(x+2)+(x-1)(x+3)=(x-1)(2x+5)\). Essayez donc d'exécuter le code Python suivant pour retrouver ce résultat :

print(factor((x - 1) * (x + 2) + (x - 1) * (x + 3)))

Vous remarquerez qu'en Python, les produits s'expriment avec *

Essayez maintenant d'obtenir à l'aide de Python le résultat que vous aviez trouvé manuellement pour factoriser A.

Développement

Pour développer une expression mathématique E, sympy utilise l'instruction expand(E).

Par exemple, on sait que \((x-1)(x-2)=x^2-3x+2\). Essayez donc d'exécuter le code Python suivant pour retrouver ce résultat :

print(expand((x - 1) * (x - 2)))

Essayez maintenant d'obtenir à l'aide de Python le résultat que vous aviez trouvé manuellement pour développer B.

Simplification

Pour simplifier une expression mathématique E, sympy utilise l'instruction simplify(E).

Par exemple, on sait que pour \(x\ne0\), on peut écrire \(\frac{2x^2+6x}{2x}=x+3\). Essayez donc d'exécuter le code Python suivant pour retrouver ce résultat :

print(simplify(2 * x**2 + 6 * x) / (2 * x))

Vous remarquez qu'en Python, les puissances peuvent s'exprimer avec **

Essayez maintenant d'obtenir à l'aide de Python le résultat que vous aviez trouvé manuellement pour simplifier C.

Résolution d'équations

Pour résoudre une équation E, sympy utilise solve(E, x).

Par exemple, on sait que la solution de l'équation \(3x+2=0\) est \(-\frac{2}{3}\). Essayez donc d'exécuter le code Python suivant pour retrouver ce résultat :

print(solve(3 * x + 2, x))

Qu'est-ce que la factorisation ?

Factoriser est souvent perçu par les élèves comme un exercice difficile. Or, il s'agit la plupart du temps d'appliquer la même technique. Voyons ensemble de quoi il s'agit.

Dans la phrase "Margaux va manger du pain et Margaux va manger du fromage", on remarque que Margaux va manger se répète car elle effectue plusieurs actions. On peut alors abréger cette phrase sans en changer le sens en écrivant :
"Margaux va manger du pain et du fromage".

On vient en fait de factoriser cette phrase en n'écrivant qu'une seule fois ce qui est commun Margaux va manger, et en regroupant ensuite le reste.

En résumé, factoriser c'est effectuer cette transformation :

(Margaux va manger du pain) + (Margaux va manger du fromage) ➡ ➡ ➡ Margaux va manger (du pain + du fromage).

On peut maintenant effectuer le même travail avec des expressions mathématiques :

Dans ce cas, le facteur commun est 5.

L'opération principale, lorsqu'on effectue un calcul, est celle que l'on fait en dernier, une fois que toutes les opérations prioritaires ont été effectuées. On remarque qu'en mathématiques, factoriser c'est passer d'une addition comme opération principale à une multiplication comme opération principale. C'est tout l'intérêt de la factorisation, qui permet de réorganiser une expression mathématique suivant une forme plus avantageuse.

⚡ Attention, il ne faut surtout pas oublier le signe +.

A RETENIR ...

Factoriser c'est transformer une somme ou une différence en un produit.

A quoi cela sert-il de factoriser ?

Factoriser sert à simplifier l'écriture d'un expression, lorsqu'il est plus simple de calculer un produit plutôt qu'une somme. Factoriser sert également à résoudre une équation de degré 2 (ou plus) grâce à la règle du produit nul. Factoriser est donc une manipulation essentielle qu'il faut absolument maîtriser.

Trouver le facteur commun

En utilisant le même raisonnement que nous venons de voir, nous trouvons facilement que \((x-1)\) est le facteur commun de l'expression \({\color{red}{(x-1)}}(x+2)+{\color{red}{(x-1)}}(5x+3)\).

Nous pouvons donc facilement factoriser en écrivant :

Il reste seulement à simplifier le reste
\((x+2)+(5x+3)=(x+5x+2+3)=(6x+5)\)

Donc le résultat de notre factorisation est :

\((x-1)(x+2)+(x-1)(5x+3)=(x-1)(6x+5)\)

Maintenant que notre expression est factorisée, elle est plus courte et plus simple à utiliser.

Apprendre à factoriser

Nous allons apprendre à factoriser en allant du plus facile au plus complexe.

Niveau ⭐

Exercice type : factoriser \(A=(x+1)(x+2)+3(x+1)\)

Le facteur commun est \((x+1)\). Il reste à regrouper les termes restants sans oublier le signe +,
donc \(A=(x+1)((x+2)+3)=(x+1)(x+2+3)=(x+1)(x+5)\)

A vous de jouer : factorisez \(B=(x-3)(4x+1)+8(4x+1)\)

\(B=(4x+1)(x+5)\)

Niveau ⭐⭐

On peut maintenant compliquer un tout petit peu l'exercice :

Exercice type : factoriser \(C=(x+1)(x+2)+(x+1)(x-5)\)

Le facteur commun est \((x+1)\). Il reste à regrouper les termes restants sans oublier le signe +,
donc \(C=(x+1)((x+2)+(x-5))=(x+1)(x+2+x-5)=(x+1)(2x-3)\)

A vous de jouer : factorisez \((x-3)(4x+1)+(2x-1)(4x+1)\)

\(B=(4x+1)(3x-4)\)

Niveau ⭐⭐⭐

On peut compliquer encore l'exercice avec le signe -. La technique est identique mais il ne faut pas l'oublier sous peine d'erreur :

Exercice type : factoriser \(C=(x+4)(x-2)-(x+4)(2x-3)\)

Le facteur commun est \((x+4)\). Il reste à regrouper les termes restants sans oublier le signe -,
donc \(C=(x+4)((x-2)-(2x-3))=(x+4)(x-2-2x+3)=(x+4)(-x+1)\)

A vous de jouer : factorisez \((x+5)(4x+1)-(2x-1)(4x+1)\)

Que se passe-t-il maintenant lorsqu'on rajoute un coefficient 3 devant l'une des expressions ?

A vous de jouer : factorisez \((x-3)(4x+1)+3(2x-1)(4x+1)\)

Niveau ⭐⭐⭐

Il est possible de rencontrer dans les calculs une expression du type \(a^5 b^3+a b^4\).

Il faut encore utiliser la méthode de recherche du facteur commun. Si on ne parvient pas à trouver ce facteur commun de tête, il suffit de décomposer les puissances en multiplications pour le voir apparaître. On obtient donc ici
\(a^5 b^3{\color{blue}{\textbf{+}}}a b^4={\color{red}{a}}\times a\times a\times a\times a\times{\color{green}{b}}\times {\color{green}{b}}\times {\color{green}{b}}{\color{blue}{\textbf{+}}}{\color{red}{a}}\times {\color{green}{b}}\times {\color{green}{b}}\times {\color{green}{b}}\times b\)

\(a\times b\times b\times b\) est donc le facteur commun. Il nous reste ensuite à regrouper les lettres restantes pour obtenir \(a^5b^3{\color{blue}{\textbf{+}}}a b^4={\color{red}{a}}\times {\color{green}{b}}\times {\color{green}{b}}\times {\color{green}{b}}\times(a\times a\times a\times a {\color{blue}{\textbf{+}}} b)=a b^3(a^4 + b)\)

A vous de jouer : factorisez \(a^2 b^2+ab^3\)

⚡ Attention, lorsque le facteur commun regroupe tous les termes, c'est qu'ils sont multipliés par 1, et IL NE FAUT PAS OUBLIER LE 1.

Exercice type : factoriser \(a b^3-a^5 b^4\)

Le facteur commun est \(a b^3\) qui regroupe toute la 1ère partie. On obtient donc \(a b^3-a^5 b^4=a b^3\times 1 -a \times a^4\times b^3 \times b=a b^3\times 1 -a b^3 \times a^4 \times b\)
donc \(a b^3-a^5 b^4=a b^3(1-a^4 b)\)

Il est facile d'oublier le 1 et c'est le principal piège de la factorisation. Ne tombez pas dedans...

A vous de jouer : factorisez \(a b^2+a^3 b^3\)
